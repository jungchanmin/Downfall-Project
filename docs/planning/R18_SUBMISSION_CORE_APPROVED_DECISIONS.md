# R18 Submission Core Approved Decisions

## 2026-07-15 Director Decision

### 공격 전조

- 주도권자는 공격 의도·전조를 제공하되 완전한 기술 정보를 항상 공개하지 않는다.
- 경험, 지식, 과거 기록, 관찰과 Memory Log로 정보 정확도가 상승한다.
- 일반 관찰 실패와 숙련 부족은 낮은 신뢰도로 표시한다.
- 최면·환각·혼란·인지 교란처럼 명확한 특수 원인이 있을 때만 거짓 전조를 확정 정보처럼 위장한다.

### 행동 취소와 주도권

- 피주도권자의 행동이 자세·부위·행동 게이트에 걸리면 행동이 취소된다.
- 주도권자는 커맨드를 일방 적용한다.
- 기본적으로 해당 합 종료 후 피주도권자에게 주도권을 넘긴다.
- `initiative_hold`·`combo_continue` 등 명시된 기술·상태 effect가 있으면 예외적으로 주도권을 유지한다.

### 역상성 카운터

- 역상성 카운터 성공 시 카운터 측이 확실하게 주도권을 획득한다.
- 성공 정도는 자세 해제, 역상태이상, 공격 역이용 등의 추가 보상만 조정한다.

### 상태이상 해제

- 상태이상 해제와 주도권 획득은 별개다.
- 해제·종료 후 동일 상태 내성이 상승한다.
- 상태 내성은 이후 재부여 저항, 지속시간 감소와 다음 주도권 경합 보정에 사용 가능하다.

### 연속 주도권 유지

- 주도권 유지 effect에는 기본 연속 상한을 둔다.
- 정확한 상한 수치는 밸런스 테스트 후 확정한다.
- 여러 유지 effect가 한 합에서 각각 추가 합을 생성하지 않는다.
- 보스·특수 기술은 명시적으로 상한을 확장할 수 있다.

### 다인전

- 기본 굴복 페이즈는 괴물 1체 대 활성 생존자 1명의 교환으로 설계한다.
- 대기 생존자는 정보 교정·상태 해제 보조·점유 해제·교대·구조 등의 제한 지원을 제공할 수 있다.
- 다대다 동시 커맨드 교환은 기본 범위에서 제외한다.
- 태그 교대전 세부 규칙은 후속 단계에서 설계한다.

### 생존자 커맨드 역할

- Service: 저부담 능동 공세·견제
- Tease: 고위험 패링·카운터
- Domination: 고위력 역전·강공·마무리
- Absorption: 평시 저효율 회복·특수상황 증폭·양날 전환
- Defiance: 수비·상태 해제·탈출·다음 경합 준비

기벽 Return은 이 기본 역할을 대체하지 않고 특정 상황과 전투 처리 단계를 강화한다.
