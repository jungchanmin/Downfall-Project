# R18 Skill Variant Approved Decisions

## 2026-07-14 Director Decision

### 기벽별 기술 변종 제공 원칙

- 하나의 기벽이 여러 기본 기술 분류와 연결될 수 있다.
- 단, 기벽 획득 직후 여러 기술 변종을 동시에 제공하지 않는다.
- `Proficiency` 단계에서는 해당 기벽의 대표 변종 1개만 해금한다.
- `Milestone` 단계에서는 두 번째 변종 또는 강한 특수 효과를 후보로 제공한다.
- 기본 기술은 기벽·로그·완화 상태와 관계없이 공용 풀에 남는다.
- 완화 중에는 기벽 전용 Return과 변종만 비활성화하며 기본 기술은 유지한다.
- 단순 수치 상위호환은 변종으로 만들지 않고 자세·부위·비용·실패 위험·전술 역할 중 하나 이상을 변경한다.
