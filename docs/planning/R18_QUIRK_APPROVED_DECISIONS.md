# R18 Quirk Approved Decisions

## 2026-07-14 Director Decisions

### 괴물 성애

- 도주·기습 선택지의 완전 하드락을 제거한다.
- 도주·기습 선택 시 정신력 또는 자제 판정을 요구한다.
- 성공하면 선택을 수행한다.
- 부분 성공은 스트레스·지연·침식 등의 대가 후 수행할 수 있다.
- 실패하면 정면 대치로 전환한다.
- 괴물 조우 스트레스 면제 Return은 유지한다.
- 파티 전체 선택을 한 NPC가 자동 봉쇄하지 않도록 한다.

### 영구 기벽의 일시 억제

- 기벽, 외형 변화와 메모리 로그는 유지한다.
- 지정된 스탯 Risk 또는 사건 가중치만 일시 억제할 수 있다.
- Risk가 억제되는 동안 직접 연결된 Return도 함께 억제한다.
- 선택지 잠금과 정체성 변화의 억제 여부는 기벽별로 정의한다.
- 영구 기벽 제거 또는 무료 버프 수단으로 사용할 수 없다.

### Return 강도

- 현재 단계에서는 선택지·기술·이벤트 해금 방향만 확정한다.
- 실제 수치 보상은 R18 Skill Catalog, Combat System, Memory Log 연동 후 밸런스 단계에서 확정한다.
