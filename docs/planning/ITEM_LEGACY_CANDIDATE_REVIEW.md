# Legacy Item Candidate Review

## 1. 목적

이 문서는 `/mnt/data/DownFall.xlsx`의 `장비정보` 시트에 존재하는 레거시 아이템 후보를 현행 `MECH_Resource_System`과 `ITEM_System_Overview` 기준으로 분류한다.

원본 수치와 효과는 정본으로 사용하지 않는다. 이 문서는 세부 Item DB 작성 전 후보 풀을 정리하는 검토 자료다.

## 2. 판정 코드

| 코드 | 의미 |
|---|---|
| 유지 | 이름과 기능 방향을 세부 DB 후보로 유지 |
| 수정 | 이름은 유지 가능하나 효과·분류를 전면 재설계 |
| 이동 | Item DB가 아닌 Entity, Event, Narrative, Skill 등 다른 문서가 소유 |
| 통합 | 유사 후보를 하나의 아이템군 또는 환산 자원으로 통합 |
| 폐기 | 현행 시스템과 충돌하거나 관리 복잡도 대비 가치가 낮음 |
| 보류 | 콘셉트 가치가 있으나 별도 검토가 필요함 |
