
# 📢 통보형 이벤트: 얼어붙은 이성 (Var 1)

## ⚙️ 1. 메타데이터
```yaml
이벤트_ID: "EV_NOTI_SelfImprove_Mind_001"
이벤트_분류: "NPC_단독"
발생_페이즈: ["아침"]
발생_장소: ["아지트"]
```

## 🎯 2. 발동 조건
```yaml
주체_조건(NPC_A): 
  - "#Stat_스트레스_>=5"
대상_조건(NPC_B): ["None"]
환경_및_연계_조건: ["None"]
```

## 📖 3. 상황 내러티브
- **이벤트 요약**: 외부 자극을 차단하고 벽에 기대어 호흡을 통제함.
- **출력 텍스트**:
  > 아침의 한기가 거실 바닥에 무겁게 깔려 있습니다. 소란스러운 다른 생존자들을 등진 채, **[NPC_A]**는 습기가 찬 차가운 벽면에 이마를 기대고 앉아 있습니다. 
  > 
  > '하나, 둘, 셋...' 
  > 
  > 규칙적이고 느릿한 숨소리만이 그 주변을 맴돕니다. 공포로 점철된 바깥 세상의 감각을 철저히 차단하고, 오직 자신의 심장 박동에만 의식을 집중합니다. 차가운 돌의 감촉이 부서져가는 이성을 단단히 붙잡아 맵니다.

## 📉 4. 자동 결과 및 수치 변동
- **스탯 변동**:
  - `[NPC_A]` : `#Stat_정신력_+1`, `#Stat_스트레스_-1`

  > 
  > 입술이 미세하게 달싹입니다. 사태 이전의 자신의 이름, 이곳에 오기 전의 삶, 그리고 아직 자신이 살아있다는 명백한 사실들을 건조하게 읊조립니다. 거울 너머의 기괴한 환영이 고개를 들기 전에, 스스로의 자아를 현실의 닻에 강제로 매어두는 고독한 의식입니다.
