# 아이템 체계 개요

## 1. 목적

이 문서는 세부 아이템 목록을 작성하기 전에 Downfall 아이템 데이터의 큰 분류와 문서 소유권을 정의한다.

세부 아이템 이름, 수치, 획득처와 밸런스는 후속 DB에서 관리한다.

## 2. 상위 분류

| 상위 분류 | 하위 분류 | 관리 방식 |
|---|---|---|
| **장착 장비** | 무기 / 보호 장비 / 유틸리티 장비 | 캐릭터 장착 또는 휴대 |
| **사용 아이템** | 의료품 / 전술품 / 기호품 / 특수 소비품 | 사용 시 소모 또는 횟수 감소 |
| **중요 물품** | 열쇠 / 문서 / 증표 / 유품 / 퀘스트 물품 | 별도 목록 또는 진행 플래그 |
| **유물** | 지속형 / 장착형 / 능동형 / 충전형 / 소모형 / 저주형 | 아이템별 규칙 |

## 3. 장착 장비

### 3.1 무기

- 근접 무기
- 원거리 무기
- 투척·폭발 무기
- 임시 무기

무기의 공격 규칙, 거리, 탄약 소비와 전투 효과는 Combat System이 소유한다.

### 3.2 보호 장비

- 일반 방어 장비
- 환경 보호 장비
- 특수 보호 장비

방어 장비는 단순 방어력뿐 아니라 가스, 감염, 화상, 오염과 기타 환경 위험 대응에 사용될 수 있다.

### 3.3 유틸리티 장비

유틸리티 장비는 특정 행동 하나만 수행하는 열쇠가 아니다.

- 관련 판정 보정
- 특별 행동 또는 선택지 해금
- 이벤트 조건 충족
- 장비 사용 횟수 또는 내구도 소비

예시 기능군은 탐색, 관측, 통신, 수리, 의료, 이동 보조다. 실제 아이템명은 DB 작성 전까지 확정하지 않는다.

## 4. 사용 아이템

### 4.1 의료품

| 하위 분류 | 정의 |
|---|---|
| **일반 의료품** | 비전투 상황에서 누구나 사용할 수 있는 기본 치료 |
| **전문 의료품** | 높은 효과를 가지며 간호 능력 또는 조건을 요구 |
| **응급 의료품** | 전투 중 즉시 사용할 수 있는 응급처치 |
| **상태이상 치료품** | 특정 상태이상 제거 또는 완화 |

### 4.2 전술품

전투, 탐색, 탈출과 교란에 범용적으로 사용할 수 있는 일회성 아이템.

아이템 설명은 허용 가능한 대표 용도를 제시하되, 이벤트가 합리적인 창의적 사용을 허용할 수 있다.

### 4.3 기호품

스트레스·컨디션 회복, NPC 취향, 관계 이벤트와 물물교환에 활용한다.

### 4.4 특수 소비품

일반 분류로 설명하기 어려운 의식, 환경, 세력 또는 특수 사건 대응 아이템.

## 5. 중요 물품

- 기본적으로 일반 인벤토리 슬롯을 차지하지 않는다.
- 이벤트 또는 루트의 조건으로 사용한다.
- 이벤트가 명시하지 않는 한 임의로 소비되지 않는다.
- 획득 출처와 연결 이벤트를 반드시 기록한다.

## 6. 유물

### 6.1 공통 정의

유물은 일반 장비보다 강력하며 핵심 세계관과 연결된 특수 아이템이다.

유물은 전부 귀속·무제한 사용으로 고정하지 않는다. 각 유물이 사용 방식, 귀속 여부, 거래 가능 여부와 부작용을 개별 정의한다.

### 6.2 사용 방식

- 지속형
- 장착형
- 능동 사용형
- 충전형
- 소모형
- 저주형

하나의 유물이 둘 이상의 사용 태그를 가질 수 있다.

## 7. 공통 데이터 필드

세부 Item DB는 최소한 다음 필드를 가진다.

```yaml
id:
name:
category:
subcategory:
description:
acquisition:
usage:
effect:
cost:
durability_or_charges:
requirements:
trade_rule:
related_lore:
status:
```

### 7.1 필드 원칙

- `effect`: 아이템 자체의 기능
- `cost`: 탄약, 사용 횟수, 내구도 또는 기타 비용
- `requirements`: 간호·제작 능력, 상태 또는 이벤트 조건
- `acquisition`: 장소명이 아니라 실제 Event·Reward 데이터와 연결 가능
- `related_lore`: 유물과 중요 물품에 우선 적용

## 8. 인벤토리 연결

- 아지트에서는 공용 창고를 사용한다.
- 외출 전 캐릭터별 장착 장비와 휴대품을 배정한다.
- 외출 중에는 배정한 아이템과 현장에서 획득한 아이템만 사용할 수 있다.
- 장착 슬롯 수, 공용 가방 슬롯과 휴대 한도는 후속 Inventory 문서에서 확정한다.
