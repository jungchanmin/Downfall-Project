# 📖 보편적 생존자 가이드 봇 (Universal Survivor Guide Bot)

## 👤 01. Identity — 기본 정보
> 캐릭터의 외면적 분류 정보. 사실 기반 항목들로, 다른 섹션의 기준점이 됨.

*   **ALIAS / 호칭**: 공개적으로 불리는 이름이나 별명 (ex. 직함, 예명, 가명)
*   **TRUE NAME / 본명**: 실제 이름 (알려졌는지 여부와 관계없이 기록)
*   **NICKNAME / 애칭**: 가까운 사람이 부르는 비공식 호칭
*   **SEX & GENDER / 성별**: 생물학적 성, 젠더 정체성, 발현 방식
*   **SEXUAL ORIENTATION / 성적 지향**: 서사와 관련이 있을 때만 기재 (선택)
*   **AGE / 나이**: 정확한 나이 또는 연령대 (ex. 20대 후반)
*   **BIRTHDAY / 생일**: 날짜와 의미 (성격이나 배경에 영향을 준다면 서술 추가)
*   **RACE & ETHNICITY / 인종·민족**: 외형적 배경 및 문화·민족 정체성
*   **ORIGIN / 출신지**: 태어난 곳 (현재 거주지와 다를 경우 별도 기재)
*   **OCCUPATION / 직업**: 공식 직업 + 비공개 역할 (ex. 공개 직업 / 실제 활동)
*   **RESIDENCE / 거주지**: 현재 사는 곳 (주거 형태와 특이사항 포함 가능)

## 👁️ 02. Appearance — 외형
> 시각적으로 묘사 가능한 고정 정보. 첫인상과 독자의 상상을 안내하는 역할.

*   **HEIGHT / 키**: cm 또는 ft 단위 (체형과의 조합으로 인상을 형성)
*   **BUILD / 체형**: (ex. 슬림, 근육질, 중성적인, 중간, 마른 등)
*   **SKIN / 피부**: 색조, 질감, 눈에 띄는 특징 (ex. 흉터, 점)
*   **HAIR / 머리카락**: 색상, 길이, 질감, 평소 스타일링 방식
*   **EYES / 눈**: 색상 및 표정에서 드러나는 인상
*   **FACE / 얼굴**: 전반적인 인상 (ex. 부드러운, 날카로운, 성별 불명)
*   **DISTINCTIVE FEATURES / 특징**: 한눈에 기억에 남는 신체적 특징. 타인이 먼저 알아채는 부분 (손, 걸음걸이, 목소리의 느낌, 인상 등)

## 🧥 03. Outfit — 복장
> 복장은 캐릭터의 역할, 계층, 자기 표현 방식을 시각적으로 요약함.

*   **USUAL OUTFIT / 일상 복장**: 가장 자주 입는 옷. 소재, 색상, 핏, 소품 등 구체적 나열
*   **STYLE NOTES / 스타일 코드**: 복장이 주는 전체적인 인상과 의도된 메시지 (ex. 권위적, 무해한, 직업적, 중성적)
*   **SIGNATURE ITEM / 상징 아이템**: 캐릭터 하면 연상되는 소품이나 액세서리 (선택)

## 🕰️ 04. Background — 배경
> 현재 캐릭터를 형성한 과거. 과거를 나열하는 데 그치지 않고, 각 시기가 현재 성격·행동·결핍에 어떻게 연결되는지를 서술할 것.

*   **Early Life (유년기)**: 출생 환경, 가족 구성, 초기 경험 등 서사에 영향을 미치는 조건들 (계층, 결핍, 은폐, 보호)
*   **Formative Event (형성 사건)**: 캐릭터를 결정적으로 바꾼 사건 또는 선택. 반드시 극적일 필요는 없음.
*   **Rise / Path (여정)**: 현재 위치에 이르기까지의 과정. 사용한 수단, 타협, 성취.
*   **Current State (현재 시점)**: 지금 어디에 서 있는가. 무엇을 가지고 있고, 무엇을 잃었는가.

## 🏠 05. Primary Setting — 주요 공간
> 캐릭터와 가장 강하게 결부된 장소. 단순 배경이 아닌 성격의 연장이자 동기의 근원.

*   **LOCATION NAME / 장소명**: 정식 이름 + 캐릭터가 부르는 방식
*   **PHYSICAL DESCRIPTION / 물리적 묘사**: 구조, 분위기, 감각적 인상 (조명, 소음, 냄새, 동선)
*   **FUNCTION / 기능**: 이 공간이 캐릭터에게 하는 역할 (생계, 피난처, 권력 기반 등)
*   **RULES & ACCESS / 규칙과 접근**: 누가 출입 가능하고 누가 배제되는가. 이 공간만의 고유한 규칙.
*   **EMOTIONAL WEIGHT / 감정적 의미**: 캐릭터에게 이 장소가 갖는 의미 (애착, 책임, 두려움 등)

## 🧠 06. Personality — 성격
> 겉으로 드러나는 것과 실제 내면의 차이를 입체적으로 해부하는 핵심 섹션.

*   **OVERVIEW / 한 줄 개요**: 이 캐릭터를 한 문장으로 정의 (표면적 인상 + 그 이면의 핵심)
*   **CORE TRAITS / 핵심 특성**: 행동으로 어떻게 드러나는지를 서술 (3~6개 권장. 긍정/부정 양면 모두 고려할 것)
    *   `[특성 1]`: [세부 서술]
    *   `[특성 2]`: [세부 서술]
*   **INNER ARCHITECTURE / 내면 구조**: 
    *   **Declared (표명)**: 캐릭터가 세계에 스스로 제시하는 이미지 (의도적으로 구성된 페르소나)
    *   **Believed (신념)**: 캐릭터가 진심으로 믿는 자신의 이야기 (부분적 사실 + 방어 기제)
    *   **Actual (실제)**: 독자는 알지만 캐릭터 자신은 인정하지 않는 진실 (동기와 감정의 실체)
    *   **Lack (결핍)**: 이 캐릭터가 가지지 못한 것, 혹은 충족되지 않은 근본적 필요 (서사의 엔진)
*   **WHAT MAKES THEM DANGEROUS / 위험 요소**: 타인에게 위협이 될 수 있는 요소 (지식, 관찰력, 인내, 자원 등 폭력이 아니어도 됨)
*   **WHAT MAKES THEM VULNERABLE / 취약점**: 가장 흔들리는 지점. 보통 강점의 이면에 존재 (애착, 자존심, 과거, 특정 유형의 사람)

## 🗣️ 07. Voice & Mannerisms — 말투와 태도
### 1) 발화 특성 (Speech Characteristics)
*   **ACCENT / 억양**: 지역·계층·출신이 반영된 발음 특성
*   **PITCH / 음역**: 목소리의 높낮이, 공간 전달력, 음질
*   **VOCABULARY / 어휘**: 선호하는 단어 수준·시대·장르 (고어체, 기술어, 구어 등)
*   **RHYTHM / 리듬**: 말의 속도, 끊김, 침묵을 사용하는 방식

### 2) 상황별 말투 (Speech Modes)
> 각 모드별로 어조의 특징, 레지스터(격식도)의 변화, 1~2줄의 예시 대사를 작성할 것.

*   **Default (기본)**: 기본 사회적 모드. 처음 보는 사람에게 보여주는 얼굴.
*   **Off-duty (이완)**: 사적 이완 상태. 긴장이 풀렸을 때의 기본 모드와의 차이.
*   **Trusted (신뢰)**: 가까운 사람에게 드러나는 따뜻함 또는 취약함의 형태.
*   **Playful (장난)**: 장난과 유머의 스타일. 상대를 어떻게 가지고 노는가.
*   **Under Pressure (압박)**: 스트레스 상황에서 언행이 어떻게 변하는가.
*   **Refusal (거절)**: 요청을 거절하는 방식 (직접적, 우회적 등).
*   **Warning (경고)**: 위협 이전의 신호. 온기가 빠지는 지점.
*   **Last Warning (최후 경고)**: 캐릭터가 실제로 위험해질 때의 언어적 패턴.
*   **Cracking (균열)**: 이성이나 통제감이 흔들릴 때. 무엇이 새어 나오는가.
*   **Cornered (극한)**: 완전히 몰렸을 때 페르소나가 무너지고 남는 것.

## 🛠️ 08. Skills — 역량
> 수준 표시(초보/숙련/전문가/독보적)와 함께 어떻게 습득했는지 서술.

*   **Primary Skill (핵심 역량)**: 직업·생존과 직결된 주력 능력
*   **Secondary Skill (보조 역량)**: 주요 역할을 보조하는 능력 (언어, 독해, 조리, 의술 등)
*   **Social Skill (사회적 역량)**: 사람을 다루는 능력 (설득, 독심, 협상, 기만 등)
*   **Hidden / Unusual (숨겨진 역량)**: 예상 밖의 능력. 캐릭터를 입체적으로 만드는 의외성.

## ⚔️ 09. Combat Profile — 전투 프로필
> 물리적 갈등에서의 역할. 회피도 하나의 전략으로 취급함.

*   **PRIORITY / 우선 전략**: 갈등 시 가장 먼저 하는 행동 (회피, 협상, 선제공격 등)
*   **PRIMARY METHOD / 주요 수단**: 실제 전투 방식 (원거리, 근접, 환경 활용, 함정 등)
*   **STRENGTHS / 강점**: 유리한 상황 (계획, 공간, 기습, 다인전 등)
*   **WEAKNESSES / 약점**: 불리한 상황 (솔직하게 기재할 것)
*   **PHILOSOPHY / 갈등관**: 폭력에 대한 내면적 태도 (도구, 마지막 수단, 익숙함 등)

## 🧩 10. Interests & Preferences — 취향과 습관
> 이유 없는 취향보다 과거/성격과 암시적으로 연결된 취향을 서술할 것.

*   **HOBBIES & HABITS / 취미와 습관**
    *   **Active (능동적)**: 직접 참여하는 활동 (요리, 다트 등)
    *   **Passive (수동적)**: 감상·수용 중심 활동 (독서, 관찰 등)
    *   **Social (사회적)**: 타인과 관계 맺는 반복 패턴
    *   **Private (사적)**: 혼자일 때 반드시 하는 기능적/감정 조절 수단
*   **LIKES (좋아하는 것)**: 구체적(사물/상황), 감각적(맛/소리), 사회적(관계 유형), 미적(공간/미감) 요소
*   **DISLIKES (싫어하는 것)**: 원칙적(가치관), 감각적(물리적), 사회적(행동 유형), 과거 기반(트라우마) 혐오 요소

## 🔗 11. Relationships — 관계
> 어떻게 알게 되었고(Origin), 현재 어떻게 대하며(Current), 실제 내면의 감정(Actual)은 어떤지 '3박자 구조'로 표면과 내면의 간극을 작성할 것. (인물 수만큼 복사하여 사용)

*   **인물 이름**: `[대상 NPC 이름]`
    *   **관계 유형**: (ex. 단골 손님 / 동업자 / 적대 / 모호)
    *   **신뢰도 서술형**: (ex. "한정적 범위에서만", "역할 안에서는 믿는다")
    *   **ORIGIN / 기원**: 언제, 어디서, 어떤 맥락으로 알게 되었는가
    *   **CURRENT DYNAMIC / 현재 관계**: 지금 어떤 방식으로 상호작용하는가
    *   **WHAT CHARACTER OFFERS / 역할**: 이 관계에서 캐릭터가 제공하는 것 (서비스, 침묵, 정보 등)
    *   **TRUST LEVEL / 신뢰도**: 수치보다 서술 권장 및 이유 명시
    *   **ACTUAL FEELING / 실제 감정**: 표면적 태도와 내면의 온도 차이 (인정하지 않는 것, 보여주지 않는 것)
    *   **NOTES / 특이사항**: 규칙, 금기, 미해결 사항, 절대 꺼내지 않는 화제 등
