# 버려진 폐가 상세 설정

## 1. 기본 정보

- **공식 명칭**: 버려진 폐가
- **영문명**: Abandoned Cabins
- **위험도**: 비교적 안전
- **중심성**: 중심부 근처
- **세력 영향**: 방랑자, 비정규 적대 집단
- **ConfigureBook 원고**: `ConfigureBook/Place/Abandoned_Houses.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

버려진 폐가는 과거 펜션 단지였던 폐허다. 생존자가 숨기 좋은 공간이지만, 같은 이유로 납치꾼, 부랑자, 적대 생존자에게도 유리하다.

이 장소의 핵심 위험은 괴물보다 인간형 위협, 함정, 납치, 실종 전조에 있다. 비교적 안전한 중심부 근처라는 위치와 달리 실제 탐색 중에는 등 뒤를 의심하게 만드는 장소로 설계한다.

## 3. Import 기준

이 문서의 이상현상은 항상 발생하는 고정 사건이 아니라 이벤트 제작 시 사용할 수 있는 후보 목록이다.

자원 획득 후보와 등장 가능 괴물·세력 후보는 현재 정본 DB와 대조가 끝나지 않았으므로 본문에 확정 등록하지 않는다.

- **자원 후보**: TODO — Resource System 또는 Item DB 정리 후 등록
- **괴물/세력 후보**: TODO — Monster DB 및 Entity 문서와 대조 후 등록

## 4. 세부 구역

### 4.1 주택가 골목 및 외부

#### 깨진 유리 골목

- **공간 성격**: 은밀한 이동과 소음 함정의 골목
- **상호작용 후보**: 은밀한 이동, 매복 탐지, 소음 제거, 추적 회피
- **이상현상 후보**: 골목 끝에서 방금 지나간 자신의 뒷모습을 목격함, 유리 조각 소리가 비정상적으로 크게 울림

#### 낡은 현관 포치

- **공간 성격**: 유인용 보급품과 함정의 입구
- **상호작용 후보**: 함정 해제, 가택 침입, 외부 망보기
- **이상현상 후보**: 먼지가 사람 형태로 흔들의자에 앉았다가 사라짐, 전선 없는 초인종이 스스로 울림

### 4.2 폐가 내부 1층

#### 거실

- **공간 성격**: 납치꾼의 유인 흔적과 실종자 기록
- **상호작용 후보**: 장비 수색 후보, 부품 수집 후보, 납치꾼 이동 경로 추적
- **이상현상 후보**: 커튼 뒤 실루엣을 걷어내면 벽면 자체가 사람 얼굴처럼 솟아 있음

#### 어두운 주방

- **공간 성격**: 화학 약품과 장기 보관 공포의 공간
- **상호작용 후보**: 화학 재료 후보 조사, 독극물 함정 제작 후보, 약물 제조 흔적 조사
- **이상현상 후보**: 가스레인지의 푸른 불길이 손 모양으로 일렁임, 배수구에서 `살려줘`라는 속삭임이 기포와 함께 올라옴

### 4.3 밀실 및 은신처

#### `교육`의 방

- **공간 성격**: 감금, 세뇌, 구출 이벤트 후보지
- **상호작용 후보**: 갇힌 동료 구출, 세뇌 기록물 조사, 해독제 후보 수색
- **이상현상 후보**: 녹음기를 틀지 않아도 `나는 복종한다`는 중첩된 목소리가 메아리침, 벽지 뒤의 수많은 눈동자

#### 천장 위 다락

- **공간 성격**: 기습형 위협과 둥지 후보지
- **상호작용 후보**: 기습 방어, 고가치 부품 후보 획득, 위쪽 공간 정찰
- **이상현상 후보**: 천장 들보가 갈비뼈처럼 수축과 이완을 반복함, 자신의 그림자가 괴물 형상으로 변해 내려다봄

#### 전리품 지하실

- **공간 성격**: 납치 피해자 분류와 거래 단서가 남은 지하 공간
- **상호작용 후보**: 노예 명부 탈취, 다른 세력 거래 단서 포착, 대규모 물자 후보 확인
- **이상현상 후보**: 빈 철장에서 쇠창살을 흔드는 금속음이 계속 들림, 내려갈 때마다 계단 수가 늘어나 지상으로 돌아가기 어려워짐

## 5. 이벤트 제작 방향

- 은신과 기습
- 납치 또는 실종 전조
- 방랑자와 비정규 적대 집단의 경계가 흐려지는 사건
- 빈집 내부의 흔적 조사
- 비교적 안전한 지역에서 발생하는 인간형 위협

## 6. Import 메모

ConfigureBook 원고의 주요 공간과 이상현상 후보를 반영했다. 단, 특정 약품, 장비, 괴물명, 세력 거래 구조는 현행 Resource/Monster/Entity DB 대조 전까지 정본화하지 않는다.
