# 버려진 폐가 상세 설정

## 1. 기본 정보

- **공식 명칭**: 버려진 폐가
- **영문명**: Abandoned Cabins
- **위험도**: 비교적 안전
- **중심성**: 중심부 근처
- **세력 영향**: 방랑자, 비정규 적대 집단
- **ConfigureBook 원고**: `ConfigureBook/Place/Abandoned_Houses.md`
- **Import 상태**: Reviewing

## 2. 역할

버려진 폐가는 과거 펜션 단지였던 폐허다. 생존자가 숨기 좋은 공간이지만, 같은 이유로 납치꾼, 부랑자, 적대 생존자에게도 유리하다.

## 3. 세부 구역

- 폐가 내부
- 외부 마당

## 4. 이벤트 제작 방향

- 은신과 기습
- 납치 또는 실종 전조
- 방랑자와 비정규 적대 집단의 경계가 흐려지는 사건
- 빈집 내부의 흔적 조사
- 비교적 안전한 지역에서 발생하는 인간형 위협

## 5. 원고 반영 메모

ConfigureBook 원고의 폐허 세부 구조와 기습 시드는 후속 Import 패스에서 정리한다.
