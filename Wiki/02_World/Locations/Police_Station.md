# 경찰서 상세 설정

## 1. 기본 정보

- **공식 명칭**: 경찰서
- **영문명**: Police Station
- **위험도**: 위험
- **중심성**: 중간 지대에서 외곽 사이
- **세력 영향**: 악마 조직
- **ConfigureBook 원고**: `ConfigureBook/Place/PoliceOffice.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

경찰서는 과거 치안 중심지였으나 현재는 악마 조직이 점거한 무기 확보 지역이다. 화력 보상은 크지만 감금, 세뇌, 추격과 괴생명체 난입 위험이 공존한다.

이 장소의 핵심 테마는 `거짓 보호`, `위장된 치안`, `납치와 교육`, `무기고 보상`, `외부 괴물 압박`이다. 구체 고유 NPC, 약물, 무기 종류와 보스전 조건은 후속 DB 대조 전까지 후보로만 취급한다.

## 3. Import 기준

이 문서의 이상현상과 위협 요소는 항상 발생하는 고정 사건이 아니라 이벤트 제작 시 사용할 수 있는 후보 목록이다.

자원 획득 후보, 무기·약물명, 고유 NPC, 구체 괴물명과 전투 조건은 현재 정본 DB와 대조가 끝나지 않았으므로 본문에 확정 등록하지 않는다.

- **자원 후보**: TODO — Resource System 또는 Item DB 정리 후 등록
- **괴물/전투 후보**: TODO — Monster DB 및 Combat 문서와 대조 후 등록
- **세력/NPC 후보**: TODO — Entity 문서와 대조 후 등록

## 4. 세부 구역

### 4.1 외부 및 로비

#### 경찰서 주차장

- **공간 성격**: 바리케이드, 순찰차, 외부 괴물 압박이 드러나는 진입부
- **상호작용 후보**: 엄폐 이동, 차량 내부 조사, 외부 위협 유인 후보
- **이상현상 후보**: 전원 없는 순찰차 경광등이 회전하며 사이렌 대신 비명 같은 소리를 냄
- **보류 항목**: 탄약·차량 부품 획득은 Resource/Item DB 대조 후 등록

#### 접수 로비 및 민원실

- **공간 성격**: 경찰 제복과 친절한 안내로 생존자를 안심시키는 거짓 안전지대
- **상호작용 후보**: 악마 조직과 접촉, 평판 작업, 임시 휴식 제안 수락 여부 선택
- **위협 후보**: 안전한 숙소와 식사 제안을 수락하면 격리·교육 구역으로 이어지는 함정 이벤트가 발생할 수 있음
- **보류 항목**: 고유 간부명과 세력 호감도 변화량은 Entity/Relationship 문서 대조 후 등록

### 4.2 행정 및 보급 구역

#### 증거물 보관실

- **공간 성격**: 압수품, 무기, 잠긴 캐비닛, 소음 트리거가 결합된 고위험 보급 구역
- **상호작용 후보**: 고가치 물자 후보 루팅, 특수 장비 후보 확보, 잠긴 캐비닛 해제
- **위협 후보**: 캐비닛을 강제로 열 때 발생하는 금속음이 외부 위협을 자극할 수 있음
- **보류 항목**: 무기 종류, 장비 등급, 난입 괴물은 DB 대조 후 등록

#### 서장실

- **공간 성격**: 악마 조직 지휘 인물의 사적 공간과 거래 제안 후보지
- **상호작용 후보**: 악마 조직 가입 제안, 배송 기록 조사, 고급 물자 후보 조사
- **이상현상 후보**: 방 안의 시계가 제각각 다른 속도로 흐름, 대화 중 플레이어의 그림자가 스스로 목을 조르는 형상을 취함
- **보류 항목**: 고유 NPC명, 가입 조건, 세력 보상은 Entity/Faction 문서 대조 후 등록

### 4.3 격리 및 교육 구역

#### 취조실 및 약물실

- **공간 성격**: 납치된 생존자의 자아를 무너뜨리는 격리 공간
- **상호작용 후보**: 납치된 동료 구출, 세뇌용 약물 후보 조사, 장부 수색
- **이상현상 후보**: 방음 벽지 너머에서 `나는 아무것도 아니다, 나는 도구다`라는 속삭임이 겹쳐 들림
- **보류 항목**: 약물 효과와 정신력·침식 변화량은 관련 시스템 문서 정리 후 등록

#### 유치장 및 임시 수용소

- **공간 성격**: 배송 대기 중인 생존자들이 갇힌 임시 수용 공간
- **상호작용 후보**: 수용자 해방, 수용자 가로채기, 악마 조직과 적대 또는 협력 선택
- **위협 후보**: 외부 위협이 벽을 뚫고 난입해 혼란 속 생존 판정으로 이어질 수 있음
- **보류 항목**: 인간성 상승/하락, 세력 적대 수치, 괴물 난입 조건은 후속 시스템 문서 대조 후 등록

#### 무기고

- **공간 성격**: 경찰 화력과 고위험 봉인이 결합된 보상 구역
- **상호작용 후보**: 최후의 무장 후보, 대규모 탄약 후보, 봉인 해제 시도
- **위협 후보**: 내부에 감염·침식된 전직 경찰관들이 갇혀 있어 개방 시 보스전급 전투 후보가 발생할 수 있음
- **보류 항목**: 구체 무기명, 보스전 조건, 침식 괴물명은 Monster/Combat DB 대조 후 등록
