# 성당 상세 설정

## 1. 기본 정보

- **공식 명칭**: 성당
- **영문명**: Cathedral
- **위험도**: 안전
- **중심성**: 중심부 근처
- **세력 영향**: 중립 지대
- **ConfigureBook 원고**: `ConfigureBook/Place/Cathedral.md`
- **Import 상태**: Reviewing

## 2. 역할

성당은 오벵마을 안쪽의 제한적 중립 공간이다. 괴생명체의 접근이 드물고 세력 간 제한적 교환이나 접촉이 가능하지만, 지하기도실과 공동묘지에는 의식과 어머니 설정에 연결되는 불길한 전조가 남아 있다.

## 3. 세부 구역

- 정원
- 성당 홀
- 지하기도실
- 공동묘지

## 4. 이벤트 제작 방향

- 중립 거래와 세력 접촉
- 신앙, 죄책감, 고해와 광신도의 유혹
- 지하기도실의 의식 흔적
- 공동묘지의 이상현상
- 안전지대처럼 보이는 공간의 배신

## 5. 원고 반영 메모

ConfigureBook 원고의 세부 공간, 종교 상징, 지하 공간 묘사는 후속 Import 패스에서 보강한다.
