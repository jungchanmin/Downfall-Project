# 성당 상세 설정

## 1. 기본 정보

- **공식 명칭**: 성당
- **영문명**: Cathedral
- **위험도**: 안전
- **중심성**: 중심부 근처
- **세력 영향**: 중립 지대
- **ConfigureBook 원고**: `ConfigureBook/Place/Cathedral.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

성당은 오벵마을 안쪽의 제한적 중립 공간이다. 괴생명체의 접근이 드물고 세력 간 제한적 교환이나 접촉이 가능하지만, 지하기도실과 공동묘지에는 의식과 어머니 설정에 연결되는 불길한 전조가 남아 있다.

성당의 안전은 종교적 성스러움만으로 설명하지 않는다. 지하 금기 구역의 비정상적 기운, 세력 간 불가침, 의식적 중심부 제어가 겹쳐 만들어진 불안정한 안전지대다.

## 3. Import 기준

이 문서의 이상현상은 항상 발생하는 고정 사건이 아니라 이벤트 제작 시 사용할 수 있는 후보 목록이다.

자원 획득 후보와 등장 가능 괴물·세력 후보는 현재 정본 DB와 대조가 끝나지 않았으므로 본문에 확정 등록하지 않는다.

- **자원 후보**: TODO — Resource System 또는 Item DB 정리 후 등록
- **괴물/세력 후보**: TODO — Monster DB 및 Entity 문서와 대조 후 등록

## 4. 세부 구역

### 4.1 광장 및 외부

#### 성당 앞 광장

- **공간 성격**: 중립 거래와 세력 접촉의 중심
- **상호작용 후보**: 물자 거래 후보, 정보 수집, 다른 세력 NPC 접촉
- **이상현상 후보**: 광장에 모인 사람들이 동시에 같은 방향을 응시하다가 아무 일 없었다는 듯 거래를 재개함

#### 종탑

- **공간 성격**: 마을 조망, 감시, 원거리 지원 후보지
- **상호작용 후보**: 안개 관측, 감시, 저격 지원 후보, 휴식
- **이상현상 후보**: 종을 치지 않았는데 낮은 진동이 울리고 생존자의 귀에서 피가 흐름

#### 성당 뒤 공동묘지

- **공간 성격**: 은밀한 접선과 죽음의 전조
- **상호작용 후보**: 유품 발굴 후보, 은밀한 접선, 묘비 조사
- **이상현상 후보**: 비석 이름이 생존자가 아는 지인의 이름으로 변함, 흙 아래에서 규칙적인 고동이 들림

### 4.2 대성당 내부

#### 중앙 예배당

- **공간 성격**: 휴식, 집단 의식, 제한적 안전
- **상호작용 후보**: 집단 정화 후보, 안전한 숙박 후보, 장비 수리 후보
- **이상현상 후보**: 스테인드글라스 속 성인들이 괴물을 경배하는 모습으로 변함

#### 고해성사실

- **공간 성격**: 비밀 폭로, 죄책감, 정신 회복과 유혹
- **상호작용 후보**: 개인 고백, 정신력 회복 후보, 은밀한 대화
- **이상현상 후보**: 칸막이 너머의 존재가 인간이 아님을 직감하지만 조언은 치명적으로 정확함

#### 주 제단

- **공간 성격**: 관리자 NPC 접촉, 로어 감정, 정적의 중심
- **상호작용 후보**: 관리 인물과 거래, 안전 구역의 진실 탐문, 로어 아이템 감정 후보
- **이상현상 후보**: 제단 근처에서 모든 소음이 완전히 차단되는 절대 정적

#### 성가대석

- **공간 성격**: 감시, 은닉, 오르간 이상현상
- **상호작용 후보**: 매복, 감시, 물자 은닉
- **이상현상 후보**: 밤마다 파이프 오르간이 비명과 유사한 선율을 스스로 연주함

### 4.3 관리 및 금기 구역

#### 성물 안치실

- **공간 성격**: 성물과 무기고가 겹친 보관 구역
- **상호작용 후보**: 장비 구입 후보, 귀중품 교환 후보, 보관품 조사
- **이상현상 후보**: 성수가 피로 변함, 은제 십자가가 손길에 뜨거워짐

#### 성당 도서관

- **공간 성격**: 오벵마을 역사, 고문서, 로어 획득지
- **상호작용 후보**: 로어 획득, 지도 해독, 고대 언어 연구 후보
- **이상현상 후보**: 책 사이에 눈동자가 나타남, 책장이 생존자의 미래 파멸을 묘사함

#### 지하실 입구

- **공간 성격**: 금기 구역의 관문
- **상호작용 후보**: 경비 돌파 시도, 문틈으로 엿듣기, 봉인 상태 조사
- **이상현상 후보**: 문 가까이에 서기만 해도 침식 관련 전조가 발생함, 문 너머 심장 박동이 성당 전체를 흔듦

### 4.4 성당 지하 금기 구역

이 구역은 성당 공통 장소 문서에 후보만 기록한다. 실제 고위험 시나리오, 보스급 존재, 침식 수치, 특수 유물 보상은 후속 이벤트·몬스터·R18 또는 로어 문서와 대조한 뒤 정본화한다.

#### 지하기도실 회랑

- **공간 성격**: 금지된 지식과 유혹
- **상호작용 후보**: 은신, 매복, 금지된 지식 조사, 목소리 듣기
- **이상현상 후보**: 사별한 가족이나 소중한 사람의 목소리가 기도실 안에서 들림

#### 망각의 공동묘지

- **공간 성격**: 실패한 그릇과 인격의 파편이 남은 지하 묘지 후보
- **상호작용 후보**: 유물 발굴 후보, 유령과의 대화 후보
- **이상현상 후보**: 유령이 과거를 말하며 감정적 동요를 일으킴

#### 통곡의 연못

- **공간 성격**: 욕망과 자기혐오를 비추는 지하 수면
- **상호작용 후보**: 검은 액체 조사 후보, 타락 전조 확인
- **이상현상 후보**: 연못에 비친 자신이 가장 저지르고 싶어 하는 악행을 수행함

## 5. 이벤트 제작 방향

- 중립 거래와 세력 접촉
- 신앙, 죄책감, 고해와 광신도의 유혹
- 지하 금기 구역의 의식 흔적
- 공동묘지와 도서관을 통한 로어 획득
- 안전지대처럼 보이는 공간의 배신

## 6. Import 메모

ConfigureBook 원고의 주요 공간과 이상현상 후보를 반영했다. 단, 고유 인물명, 보스급 존재, 자원명, 괴물명, 침식 수치 효과는 현행 DB와 대조 전까지 정본화하지 않는다.
