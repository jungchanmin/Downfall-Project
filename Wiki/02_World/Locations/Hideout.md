# 아지트 상세 설정

## 1. 기본 정보

- **공식 명칭**: 아지트
- **영문명**: Hideout
- **위험도**: 안전
- **중심성**: 중심부
- **세력 영향**: 생존자 무리
- **ConfigureBook 원고**: `ConfigureBook/Place/Azit.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

아지트는 매 회차 고정된 시작 거점이다. 플레이어와 동료들은 이곳에서 자원 분배, 휴식, 대화, 갈등, 치료, 제작, 경계와 야간 사건을 처리한다.

아지트의 안전은 절대적 보호가 아니다. 의식이 그릇 후보와 제물을 보존하기 위해 허락한 제한적 안식이며, 외부 위협 대신 자원 부족과 내부 갈등을 증폭시키는 압력솥으로 기능한다.

## 3. Import 기준

이 문서의 이상현상은 항상 발생하는 고정 사건이 아니라 이벤트 제작 시 사용할 수 있는 후보 목록이다.

자원 획득 후보와 등장 가능 괴물·세력 후보는 현재 정본 DB와 대조가 끝나지 않았으므로 본문에 확정 등록하지 않는다.

- **자원 후보**: TODO — Resource System 또는 Item DB 정리 후 등록
- **괴물/세력 후보**: TODO — Monster DB 및 Entity 문서와 대조 후 등록

## 4. 세부 구역

### 4.1 실외 및 진입로

#### 대문

- **공간 성격**: 외부 위협 차단, 경계, 침입 흔적 확인
- **상호작용 후보**: 경계 서기, 문 보강, 외부 위협 차단, 못질하여 보강하기
- **이상현상 후보**: 잠가둔 문이 아침마다 미세하게 열려 있음, 문틀에 사람의 손톱자국이 깊게 파여 있음

#### 정원

- **공간 성격**: 은신, 침입 흔적 조사, 외부 활동 공간
- **상호작용 후보**: 은신, 외부 침입 흔적 조사, 자재 확보 후보 기록
- **이상현상 후보**: 잡풀의 배치가 매일 밤 조금씩 바뀜, 바람이 없어도 풀들이 아지트 쪽으로 눕는 현상

#### 연못

- **공간 성격**: 장기 복선과 수면 이상현상 발생지
- **상호작용 후보**: 위험한 물 정화 시도, 투척 무기 연습
- **이상현상 후보**: 수면에 비친 그림자가 실제 움직임보다 늦게 반응함, 물속의 흰 손 같은 형체가 올라왔다 사라짐

#### 텃밭

- **공간 성격**: 생존과 오염이 맞닿는 생활 공간
- **상호작용 후보**: 식량 재배 후보, 음식물 쓰레기 처리, 토양 상태 조사
- **이상현상 후보**: 수확한 뿌리 끝에서 검은 액체가 흘러나옴, 밤마다 흙 속에서 젖은 소리가 남

#### 수영장

- **공간 성격**: 함정 설치와 불길한 수분 이상현상 후보지
- **상호작용 후보**: 대규모 함정 설치, 빗물 수집 후보
- **이상현상 후보**: 비가 오지 않아도 점액질이 차오름, 밤중 수영장 한가운데에 사람 실루엣이 서 있음

### 4.2 1층 공용 구역

#### 현관

- **공간 성격**: 외출과 귀환의 경계
- **상호작용 후보**: 외출 전 장비 점검, 덧문 확인, 신발 정비
- **이상현상 후보**: 거울에 비친 복도가 실제보다 길어 보임, 문밖에서 정중한 세 번의 노크가 반복됨

#### 거실

- **공간 성격**: 중앙 허브, 회의와 휴식의 중심
- **상호작용 후보**: 휴식, 작전 회의, 집단 의사결정
- **이상현상 후보**: 꺼진 TV에 생존자의 과거 기억이 노이즈로 출력됨, 벽난로 안에서 타지 않은 머리카락 뭉치가 발견됨

#### 식당

- **공간 성격**: 배식, 대화, 집단 긴장 발생지
- **상호작용 후보**: 배식, 단체 식사, 호감도 이벤트
- **이상현상 후보**: 식사 중 모든 의자가 동시에 뒤로 밀려남, 접시 위 음식이 순식간에 썩음

#### 주방

- **공간 성격**: 조리, 보관, 제작 후보 공간
- **상호작용 후보**: 조리, 도구 제작 후보, 식량 보관
- **이상현상 후보**: 칼 위치가 바뀜, 수도꼭지에서 녹물과 함께 치아가 섞여 나옴

#### 세탁실

- **공간 성격**: 의복·붕대 관리와 기이한 생활 소음 발생지
- **상호작용 후보**: 의복 수선, 붕대 세척 후보
- **이상현상 후보**: 세탁기 안에서 멈추지 않는 웅얼거림, 건조대 옷소매가 잡아당긴 듯 늘어남

#### 복도

- **공간 성격**: 이동과 매복, 공간 왜곡 후보지
- **상호작용 후보**: 이동, 소리 죽여 걷기, 매복
- **이상현상 후보**: 걸어도 다음 방이 나오지 않음, 벽지 뒤에서 심장 박동 소리가 들림

#### 작은방

- **공간 성격**: 개인 정비와 은신, 어린이 방 공포
- **상호작용 후보**: 개인 정비, 일기 쓰기, 은신
- **이상현상 후보**: 장난감 오르골이 밤마다 연주됨, 벽에 `나를 찾아줘` 낙서가 피어오름

#### 창고

- **공간 성격**: 물자 보관과 변질 후보 공간
- **상호작용 후보**: 물자 수색, 함정 제작 후보
- **이상현상 후보**: 박스 안 내용물이 과거 유품으로 변함, 창고 안의 소리가 밖으로 새지 않음

#### 지하실

- **공간 성격**: 비상 보관, 구금, 최후의 대피소
- **상호작용 후보**: 비상 식량 보관 후보, 구금, 최후의 대피
- **이상현상 후보**: 보일러 소리가 비명으로 변함, 바닥 그림자가 벽을 타고 올라옴

### 4.3 2층 및 사적 공간

#### 큰방

- **공간 성격**: 숙면, 사적 대화, 수면 중 불안
- **상호작용 후보**: 숙면, 은밀한 거래, 수면 중 기습 방어
- **이상현상 후보**: 침대 밑 젖은 발자국, 장롱 안 눈동자

#### 베란다

- **공간 성격**: 감시와 외부 신호 후보지
- **상호작용 후보**: 감시, 외부 세력과 신호 교환 후보, 스트레스 해소 행동 후보
- **이상현상 후보**: 안개 속 거대한 팔 같은 형체, 흔들의자가 혼자 격렬히 흔들림

#### 샤워실

- **공간 성격**: 오염 제거와 신체 불안
- **상호작션 후보**: 오염 제거 후보, 상처 세척
- **이상현상 후보**: 비누 거품 속에서 머리카락이 계속 나옴, 샤워 커튼 너머 실루엣

#### 화장실

- **공간 성격**: 위생과 신체 공포 후보지
- **상호작용 후보**: 위생 관리, 세면, 단독 이벤트
- **이상현상 후보**: 검은 액체, 배수구·거울·문틈을 활용한 이상현상 후보

## 5. 이벤트 제작 방향

- 자원 부족으로 인한 내부 갈등
- 야간 경계 실패와 침입 흔적
- 생존자 간 관계 변화
- 아지트가 안전하다는 믿음을 흔드는 이상현상
- 지하실, 연못, TV, 거울을 통한 장기 복선

## 6. Import 메모

ConfigureBook 원고의 주요 공간과 이상현상 후보를 반영했다. 단, 자원명과 괴물명은 현행 Resource/Monster DB 대조 전까지 정본화하지 않는다.
