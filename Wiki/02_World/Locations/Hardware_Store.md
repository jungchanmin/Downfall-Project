# 철물점 상세 설정

## 1. 기본 정보

- **공식 명칭**: 철물점
- **영문명**: Hardware Store
- **위험도**: 위험
- **중심성**: 중간 지대
- **세력 영향**: 자경단
- **ConfigureBook 원고**: `ConfigureBook/Place/SteelFort.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

철물점은 생존과 전투에 직결되는 부품, 도구, 가공 장비, 유물성 장비가 모이는 장소다. 자경단의 관리 아래 있으므로 거래, 압수, 제작 의뢰와 통제 이벤트에 적합하다.

이 장소의 핵심 테마는 `제작 자원`, `자경단 검문`, `소음에 민감한 위험`, `도구가 무기가 되는 공간`이다. 구체 자원명과 제작 수치는 후속 Resource/Item/Combat 문서 대조 전까지 후보로만 취급한다.

## 3. Import 기준

이 문서의 이상현상과 위협 요소는 항상 발생하는 고정 사건이 아니라 이벤트 제작 시 사용할 수 있는 후보 목록이다.

자원 획득 후보, 제작 효과, 괴물명, 자경단 고유 NPC와 보상은 현재 정본 DB와 대조가 끝나지 않았으므로 본문에 확정 등록하지 않는다.

- **자원 후보**: TODO — Resource System 또는 Item DB 정리 후 등록
- **제작/전투 효과**: TODO — Crafting/Combat 문서 정리 후 등록
- **괴물/세력 후보**: TODO — Monster DB 및 Entity 문서와 대조 후 등록

## 4. 세부 구역

### 4.1 외부 및 하역장

#### 검문소: 정문 셔터

- **공간 성격**: 자경단 검문과 출입 통제의 경계
- **상호작용 후보**: 통행권 제시, 뇌물 또는 물자 협상, 검문 응하기, 은밀한 침입 시도
- **위협 후보**: 검문 명목으로 물자를 빼앗기거나 약자를 압박당할 수 있음. 거부 시 폭행, 구금, 강제 추방으로 이어질 수 있음

#### 뒷마당 하역장

- **공간 성격**: 컨테이너와 자재가 쌓인 은밀한 우회·침투 공간
- **상호작용 후보**: 은밀한 진입, 대형 자재 후보 조사, 약탈자 흔적 추적
- **이상현상 후보**: 팔레트 사이에서 도움을 청하는 소리가 들리고, 확인하면 나무 조각들이 사람 형태로 뒤틀려 있음

### 4.2 철물점 내부 1층

#### 공구 진열대

- **공간 성격**: 공구, 도구, 임시 무기 후보가 집중된 고감시 구역
- **상호작용 후보**: 무기급 도구 후보 수색, 기계 부품 후보 조사, 연장 챙기기
- **이상현상 후보**: 도구를 집는 순간 진열된 망치들이 바닥을 치는 듯한 환청이 들려 공황 전조를 유발함

#### 페인트 및 화학실

- **공간 성격**: 시야가 나쁘고 화학 냄새가 강한 위험 구역
- **상호작용 후보**: 화학 재료 후보 조사, 연막·화염 계열 함정 후보 확인, 독성 흔적 조사
- **이상현상 후보**: 붉은 페인트가 살아있는 것처럼 발목을 타고 오르고, 지워지지 않는 얼룩이 낙인처럼 남음
- **보류 항목**: 낙인의 실제 효과와 괴물 유인 여부는 Monster/Event 문서 대조 후 등록

#### 서비스 카운터

- **공간 성격**: 자경단 임시 지휘소와 장부 조작 후보지
- **상호작용 후보**: 세력 퀘스트 수락 후보, 장부 조작, 자경단 교란, 압수품 조사
- **위협 후보**: 부패한 자경단 인물과 대면해 불합리한 임무나 강압적 요구를 받을 수 있음
- **보류 항목**: 고유 NPC명과 구체 임무 보상은 Entity/Quest 문서 대조 후 등록

### 4.3 창고 및 사무실

#### 대형 자재 창고

- **공간 성격**: 대량 자재와 소음 위험이 결합된 창고
- **상호작용 후보**: 바리케이드 자재 후보 조사, 발전기 부품 후보 확인, 높은 선반 정찰
- **위협 후보**: 금속 자재 소음이 발생하면 선반 위의 기습형 위협이 반응할 수 있음
- **보류 항목**: 특정 괴물명과 전투 개시 조건은 Monster/Combat 문서 대조 후 등록

#### 관리자 취조실

- **공간 성격**: 외부인을 압박하고 물건을 빼앗는 자경단 내부 처벌 공간
- **상호작용 후보**: 갇힌 생존자 구출, 기밀 문서 탈취, 자경단의 폭력 증거 확보
- **위협 후보**: 비명이나 폭행 흔적을 목격하고 개입 여부를 선택하는 인간성 시험 이벤트
- **보류 항목**: 인간성·카르마 효과는 관련 시스템 문서 정리 후 등록

## 5. 이벤트 제작 방향

- 무기와 부품 후보 획득
- 제작 또는 수리 의뢰
- 자경단의 통행 허가와 징발
- 위험한 장비 거래 또는 압수품 회수
- 전투 준비와 자원 소모의 선택
- 소음이 곧 전투 위험으로 이어지는 긴장감

## 6. Import 메모

ConfigureBook 원고의 주요 공간과 위협 후보를 반영했다. 단, 자원명, 제작 효과, 괴물명, 구체 전투 조건, 노골적 R18 성격의 강압 묘사는 공통 장소 정본에 확정하지 않는다. R18 확장은 후속 `Wiki/90_Modules/R18/`에서 분리한다.
