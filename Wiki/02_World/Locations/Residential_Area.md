# 주거단지 상세 설정

## 1. 기본 정보

- **공식 명칭**: 주거단지
- **영문명**: Residential Area
- **위험도**: 매우 위험
- **중심성**: 외곽
- **세력 영향**: 광신도
- **ConfigureBook 원고**: `ConfigureBook/Place/DownTown.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

주거단지는 사태 이전 주민들이 살던 대규모 아파트 생활권이다. A·B·C동은 서로 다른 형태의 붕괴를 보여주며, 생활 흔적·광신도 감시·괴물 서식지가 한 공간에 중첩된다.

## 3. Import 기준

세부 공간과 이상현상은 이벤트 제작 후보로 기록한다. 자원명, 괴물명, 보스전, 유물, 수치 효과와 고유 광신도 인물은 DB 대조 전까지 확정하지 않는다.

## 4. 세부 구역

### 4.1 A동: 수직의 미로

#### 피로 물든 로비
- **공간 성격**: 파손된 진입부와 의식 경고판
- **상호작용 후보**: 입구 확보, 단지 지도 조사, 주민 기록 확인
- **이상현상 후보**: 우편함에서 흐느낌이 들리고, 특정 칸 내부가 생체 조직처럼 변함

#### 비상계단 및 복도
- **공간 성격**: 좁은 복도와 반복되는 매복 지점
- **상호작용 후보**: 호실 탐색, 은밀한 이동, 기습 방어
- **위협 후보**: 좁은 구조로 인해 회피와 후퇴가 제한되는 전투 후보

### 4.2 B동: 어둠의 내장

#### 지하 주차장
- **공간 성격**: 차량 잔해와 생존자 구출 후보가 있는 암흑 구역
- **상호작용 후보**: 차량 부품 후보 조사, 매달린 생존자 구출, 상층 진입로 확보
- **위협 후보**: 다수 괴물이 결합된 대형 군집체 후보

#### 관리사무소 및 보안실
- **공간 성격**: CCTV 감시와 광신도 관측 거점
- **상호작용 후보**: 마스터키 후보 확보, 괴물 위치 파악, 붕괴 기록 조사
- **이상현상 후보**: CCTV 화면 속 플레이어 뒤에만 괴물 형상이 나타남

### 4.3 C동: 광기의 성소

#### 펜트하우스 및 옥상
- **공간 성격**: 광신도 고위 인물과 유물 후보가 집중된 성소
- **상호작용 후보**: 마을 전경 정찰, 제단 조사, 고위 광신도와 대면
- **보류 항목**: 고유 장로, 유물 보상, 보스전은 Cult/Artifact/Combat 문서 대조 후 등록

#### 주민 커뮤니티 센터
- **공간 성격**: 수영장과 운동시설이 의식·처벌 공간으로 변질된 구역
- **상호작용 후보**: 창고 후보 조사, 의식 흔적 파괴, 생존자 흔적 확인
- **이상현상 후보**: 운동기구가 스스로 작동하고 보이지 않는 존재의 발소리가 이어짐

## 5. 이벤트 제작 방향

- 일상 공간이 괴물의 둥지와 의식장으로 변질된 공포
- 좁은 아파트 복도의 매복과 실종
- 광신도 감시 체계와 제물 수색
- 주민 기록과 붕괴 과정 조사

## 6. Import 메모

원고의 A·B·C동 구조와 주요 후보를 반영했다. 구체 괴물, 유물, 자원, 보스전과 수치 효과는 정본 DB 검증 전까지 보류한다.
