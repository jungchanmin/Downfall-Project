# 주거단지 상세 설정

## 1. 기본 정보

- **위험도**: 매우 위험
- **중심성**: 외곽
- **세력 영향**: 광신도
- **ConfigureBook 원고**: `ConfigureBook/Place/DownTown.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

주거단지는 과거 주민 생활권이 괴물의 둥지와 광신도의 제물 사냥터로 변질된 장소다. A·B·C동의 수직 구조를 이용해 밀폐 전투, 실종자 구출, 감시와 의식 복선을 제공한다.

## 3. Import 기준

공간·상호작용·이상현상은 이벤트 후보로만 기록한다. 자원, 괴물명, 보스, 유물, 전투 수치는 DB 대조 전까지 확정하지 않는다.

## 4. 세부 구역

### A동: 수직의 미로

- **피칠갑된 로비**: 입구 확보, 지도 확인, 의식 경고 조사 후보
- **이상현상 후보**: 우편함에서 흐느낌이 들리거나 내부 내용물이 생체 조직으로 변함
- **비상계단 및 복도**: 호실 탐색, 은밀 이동, 좁은 복도 기습 후보
- **보류 항목**: 복도 매복 괴물과 회피 제한 규칙은 Monster/Combat 문서 대조 후 등록

### B동: 어둠의 내장

- **지하 주차장**: 차량 부품 후보 조사, 고치에 갇힌 생존자 구출, 거대 군집 위협 후보
- **관리사무소 및 보안실**: 마스터키 후보, CCTV 관측, 단지 붕괴 기록 획득
- **이상현상 후보**: CCTV에 플레이어 뒤의 존재가 비치지만 현실에는 보이지 않음

### C동: 광기의 성소

- **펜트하우스 및 옥상**: 광신도 고위 인물 거점, 마을 정찰, 유물 후보 조사
- **커뮤니티 센터**: 수영장·헬스장 의식 흔적, 광신도 훈련 공간 후보
- **이상현상 후보**: 운동 기구가 스스로 작동하며 보이지 않는 존재의 움직임을 재현함
- **보류 항목**: 광신도 장로, 보스전, 유물 보상은 Entity/Monster/Item 문서 대조 후 등록

## 5. 이벤트 제작 방향

- 생활 공간이 둥지와 제단으로 변질된 공포
- 실종자와 포로 구출
- 좁은 복도·계단 전투 후보
- 광신도 감시망과 제물 사냥
- 일상 흔적과 생체 조직의 대비

## 6. Import 메모

원고의 3개 동 구조와 핵심 후보를 반영했다. 구체 자원·괴물·광신도 인물·보상은 정본 DB 확인 전까지 보류한다.
