# 상점가 상세 설정

## 1. 기본 정보

- **공식 명칭**: 상점가
- **영문명**: Shopping District
- **위험도**: 위험
- **중심성**: 중간 지대
- **세력 영향**: 자경단, 악마 조직
- **ConfigureBook 원고**: `ConfigureBook/Place/Market_District.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

상점가는 생필품과 식량이 남아 있는 자원 밀집지다. 동시에 자경단과 악마 조직의 이권이 충돌하는 분쟁 지대이므로 탐색 보상과 인간형 위협이 함께 발생한다.

이 장소의 핵심 테마는 `보급선`, `거리 전투`, `약탈과 거래`, `소음과 빛에 반응하는 위험`이다. 구체 자원명과 괴물 특성은 후속 DB 대조 전까지 후보로만 취급한다.

## 3. Import 기준

이 문서의 이상현상과 위협 요소는 항상 발생하는 고정 사건이 아니라 이벤트 제작 시 사용할 수 있는 후보 목록이다.

자원 획득 후보, 구체 괴물 특성, 무기·식량·의약품 이름은 현재 정본 DB와 대조가 끝나지 않았으므로 본문에 확정 등록하지 않는다.

- **자원 후보**: TODO — Resource System 또는 Item DB 정리 후 등록
- **괴물 후보**: TODO — Monster DB와 대조 후 등록
- **세력 후보**: TODO — Entity 문서와 대조 후 등록

## 4. 세부 구역

### 4.1 상점가 대로 및 외부

#### 중앙 교차로

- **공간 성격**: 세력 간 충돌, 엄폐 이동, 도로 잔해를 활용한 전술 공간
- **상호작용 후보**: 세력 간 교전 목격 및 개입, 차량 잔해 조사, 엄폐 이동
- **이상현상 후보**: 신호등의 색에 따라 보이지 않는 충격이 가해짐, 전복된 버스 창문 안에서 유령들이 도움을 요청함
- **보류 항목**: 차량 연료 획득은 Resource System 정리 후 등록

#### 뒷골목

- **공간 성격**: 악마 조직의 우회로, 매복, 비밀 표식과 은닉 공간
- **상호작용 후보**: 은밀한 우회, 매복 감지, 은닉 물자 후보 조사, 조직 표식 해석
- **이상현상 후보**: 벽면 낙서가 위치를 바꿈, 그림자가 벽에 고정되지 않고 독자적으로 골목 안쪽으로 기어감

### 4.2 주요 보급 거점

#### 철물점 구획

- **공간 성격**: 상점가 내부의 공구·부품 거점 후보
- **상호작용 후보**: 부품 후보 수색, 아지트 보강 후보 물자 조사, 함정 해제
- **이상현상 후보**: 전원 없는 전동 공구가 동시에 작동함, 금속 부품들이 발치로 모여 이동을 방해함
- **소유권 주의**: 철물점의 독립 장소 정본은 `Hardware_Store.md`가 소유한다. 이 항목은 상점가 안의 연계 구역 후보로만 사용한다.

#### 대형 마트

- **공간 성격**: 자경단이 요새화한 보급 핵심 지점
- **상호작용 후보**: 자경단과 거래 후보, 식량 약탈 시도, 세력 임무 수행 후보
- **이상현상 후보**: 냉동고 안의 썩은 고기들이 하나의 살덩어리처럼 엉김, 안내방송이 죽은 점원의 목소리로 반복됨
- **보류 항목**: 식량 종류와 보급량은 Resource System 정리 후 등록

#### 약국

- **공간 성격**: 의료 물자와 환각성 이상현상 후보지
- **상호작용 후보**: 의약품 후보 조사, 조제실 금고 조사, 약품 부작용 단서 확인
- **이상현상 후보**: 약병들이 심장 박동에 맞춰 색을 바꿈, 특정 약을 집으면 부작용 환각이 먼저 발생함
- **보류 항목**: 약품명, 회복량, 부작용 수치는 Item/Medical 문서 정리 후 등록

### 4.3 특수 구역

#### 가전제품 매장

- **공간 성격**: 전자 부품, 통신 장비, 감시와 방해 장치 후보지
- **상호작용 후보**: 전자 부품 후보 조사, 통신 장비 수리 재료 후보 확인, 방해 장치 무력화 후보
- **이상현상 후보**: TV마다 다른 장소에서 죽어가는 생존자의 모습이 출력됨, 모니터 사이에서 검은 손이 장비를 낚아채려 함

#### 지하 주차장

- **공간 성격**: 어둠, 매복, 납치 피해자 임시 수용 후보지
- **상호작용 후보**: 차량 부품 후보 조사, 납치된 생존자 구출, 대규모 매복 준비
- **이상현상 후보**: 전조등이 플레이어를 따라 회전함, 주차선이 미로처럼 꼬여 출구를 찾지 못하게 함
- **보류 항목**: 괴물 유입 조건과 전투 발생 조건은 Monster/Event 문서 대조 후 등록

## 5. 이벤트 제작 방향

- 식량과 생필품 탐색
- 자경단과 악마 조직의 총격 또는 대치
- 약탈자와 일반 생존자 사이의 충돌
- 뒷골목 거래와 함정
- 자원 선택과 도덕적 비용
- 소음과 빛이 위험을 증폭시키는 거리 이벤트

## 6. Import 메모

ConfigureBook 원고의 주요 공간과 이상현상 후보를 반영했다. 단, 자원명, 회복/제작 효과, 괴물의 구체 특성, 세력 임무 보상은 현행 DB 대조 전까지 정본화하지 않는다.
