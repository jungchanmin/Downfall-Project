# 공장 상세 설정

## 1. 기본 정보

- **위험도**: 접근금지
- **중심성**: 외곽 심부
- **세력 영향**: 악마 조직
- **ConfigureBook 원고**: `ConfigureBook/Place/Factory.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

공장은 악마 조직의 핵심 병기창이자 기계 함정, 약물 실험과 자아 말소 위협이 집중된 장소다.

## 3. Import 기준

공간·위협·이상현상은 이벤트 후보다. 시체 인형, 고유 수장, 유물, 인형화 게임오버, 인간성·명성 수치는 Monster/Entity/R18/System 문서 대조 전까지 확정하지 않는다.

## 4. 세부 구역

- **환청의 벌판**: 방독 장비 점검, 정신 유지, 폐자재 조사 후보
- **위협 후보**: 공장 가스가 환청과 방향 감각 상실을 유발함
- **살육의 정문**: 자동 방어 장치, 지뢰·전류·산성 함정 후보
- **생산 라인**: 기계 과부하, 부품 후보 조사, 프레스 이상현상
- **이상현상 후보**: 프레스가 내려올 때 피가 문장을 형성함
- **약물 배양실**: 자아를 잃은 병력 후보, 배양관 파괴와 구출·처분 선택
- **유물 금고**: 고위험 보상 후보, 획득 후 공장 봉쇄 시퀀스
- **지휘 본부**: 조직 감시망 조사, 인형화 제안과 배신 선택 후보

## 5. 이벤트 제작 방향

- 병기창 침투와 자동 방어 돌파
- 약물 배양실의 인간성 선택
- 기계 소음과 함정
- 고위험 보상 후 봉쇄 탈출
- 악마 조직의 자아 말소 기술 복선

## 6. Import 메모

원고의 공장 구조와 위협 후보를 반영했다. 플레이어 인형화 게임오버, 고유 수장, 유물명, 특수 병력과 수치 효과는 미승인 설정이므로 보류한다.
