# 공장 상세 설정

## 1. 기본 정보

- **공식 명칭**: 공장
- **영문명**: Factory
- **위험도**: 접근금지
- **중심성**: 외곽 심부
- **세력 영향**: 악마 조직
- **ConfigureBook 원고**: `ConfigureBook/Place/Factory.md`
- **Import 상태**: Reviewing

## 2. 역할

공장은 악마 조직의 핵심 병기창이다. 허가받지 않은 접근은 즉각적인 전투 또는 처형 위험으로 이어지며, 기계 소음과 기름 냄새 속에서 멸망의 괴물이 출몰할 수 있다.

## 3. 세부 구역

- 생산 라인
- 창고
- 본부

## 4. 이벤트 제작 방향

- 병기창 침투
- 악마 조직 핵심 인물과의 접촉
- 폭발물과 기계 함정
- 멸망의 괴물 전조
- 대량 자원과 치명적 위험의 교환

## 5. 원고 반영 메모

ConfigureBook 원고의 생산 라인, 기계 설비와 병기창 구조는 후속 Import 패스에서 확장한다.
