# 유흥가 상세 설정

## 1. 기본 정보

- **위험도**: 매우 위험
- **중심성**: 외곽
- **세력 영향**: 악마 조직
- **ConfigureBook 원고**: `ConfigureBook/Place/NightLife_District.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

유흥가는 악마 조직이 운영하는 거래·도박·투기·납치의 거점이다. 공통 장소 정본은 범죄, 강압, 착취, 정보 거래와 위험 선택만 소유하며, 성인 전용 묘사와 결과는 R18 모듈이 소유한다.

## 3. Import 기준

이상현상과 위협은 이벤트 후보다. 고유 간부, 금화 경제, 유물, 약물, R18 장면과 수치 효과는 관련 정본 대조 전까지 확정하지 않는다.

## 4. 세부 구역

### 환락의 거리

- **네온 대로**: 통행 허가 확인, 시설 탐색, 조직원 감시 회피
- **위협 후보**: 무단 침입자는 추격·납치·강제 거래 대상으로 분류됨
- **긴 골목**: 은밀 이동, 암시장 입구 탐색, 뒷거래 목격
- **이상현상 후보**: 포스터 속 인물이 지나가는 생존자를 비웃거나 손가락질함

### 욕망의 전당

- **도박장**: 고위험 거래, 정보원 포섭, 빚진 생존자 사건 후보
- **이상현상 후보**: 카드가 남은 생존 일수나 불길한 미래를 속삭임
- **지하 투기장**: 투사 참여, 승부 조작, 전투 관찰 후보
- **주점**: 정보 수집, 밀담, 약물 함정 후보
- **감금·착취 구역**: 납치된 생존자 구출과 조직의 상품화 구조 조사 후보
- **R18 분리**: 성적 서비스·유혹·강제 장면은 `Wiki/90_Modules/R18/`에서 확장

### 권력과 거래의 심장

- **간부 접촉 시설**: 악마 조직 간부와 협상·협박·포섭 후보
- **지하 암시장**: 유물·희귀 물자 후보 거래, 사기와 강탈 위험
- **보류 항목**: 고유 간부명, 가격, 금화 수량, 유물명은 Entity/Economy/Item 문서 대조 후 등록

## 5. 이벤트 제작 방향

- 암시장 거래와 사기
- 투기장 강제 참여 또는 자발적 도전
- 악마 조직의 협박과 포섭
- 납치 피해자 구출
- 거래 비용과 인간성의 충돌

## 6. Import 메모

원고의 공통 범죄·거래·이상현상 후보를 반영했다. R18 전용 내용은 분리했으며, 고유 인물·경제·보상은 보류한다.
