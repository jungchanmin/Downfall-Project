# 선착장 상세 설정

## 1. 기본 정보

- **공식 명칭**: 선착장
- **영문명**: Dockyard
- **위험도**: 매우 위험
- **중심성**: 외곽 봉쇄선
- **세력 영향**: 없음, 무법 지대
- **ConfigureBook 원고**: `ConfigureBook/Place/Wharf.md`
- **Import 상태**: Reviewing

## 2. 역할

선착장은 물류창고와 폐선박이 남아 있어 자원 보상이 크지만, 바다와 맞닿은 괴생명체 생태계 때문에 가장 불안정한 외곽 지역 중 하나다. 탈출 시도와 구조 실패를 보여주는 장소로도 사용한다.

## 3. 세부 구역

- 해변가
- 부둣가
- 물류창고
- 광장
- 폐선박

## 4. 이벤트 제작 방향

- 대량 자원 확보 시도
- 폐선박 조사
- 탈출 루트 오판
- 바다 괴물과 이상현상
- 무법 지대의 약탈자 조우

## 5. 원고 반영 메모

`Wharf` 원고의 해상·물류·폐선박 관련 세부 시드는 후속 Import 패스에서 보강한다.
