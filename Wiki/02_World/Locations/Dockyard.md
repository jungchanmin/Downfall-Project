# 선착장 상세 설정

## 1. 기본 정보

- **위험도**: 매우 위험
- **중심성**: 외곽 봉쇄선
- **세력 영향**: 없음, 무법 지대
- **ConfigureBook 원고**: `ConfigureBook/Place/Wharf.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

선착장은 탈출이 불가능하다는 사실을 체감시키는 외곽 봉쇄선이자 고위험 물류 지역이다.

## 3. Import 기준

공간·이상현상·탈출 시퀀스는 이벤트 후보로 기록한다. 자원, 해양 괴물, 유물, 정신 피해 수치는 DB 대조 전까지 확정하지 않는다.

## 4. 세부 구역

- **컨테이너 적치장**: 미로형 이동, 광신도 매복, 장비 후보 탐색
- **이상현상 후보**: 빈 컨테이너에서 집단 두드림이 들리고 검은 바닷물이 차오름
- **대기실**: 탈출 실패 흔적, 유류품 조사, 광신도 사냥꾼 위협 후보
- **부서진 방파제**: 해안 관측, 유물 후보 탐색, 소중한 사람이 바다에서 손짓하는 환각
- **검은 모래사장**: 거대 해양 위협의 흔적과 은밀 이동 후보
- **폐선박**: 통신 시도, 고가치 물자 후보, 유물 획득 후 봉쇄되는 탈출 시퀀스 후보
- **크레인 탑**: 정찰, 저격 지점, 외부 세계에 대한 불길한 환각 후보

## 5. 이벤트 제작 방향

- 대량 물자 탐색과 바다의 시선
- 탈출 루트 실패
- 폐선박 봉쇄 탈출
- 광신도 매복
- 심해 위협의 전조

## 6. Import 메모

원고의 해안·컨테이너·폐선박 후보를 반영했다. 고유 괴물명, 유물, 정신 피해와 실제 탈출 규칙은 보류한다.
