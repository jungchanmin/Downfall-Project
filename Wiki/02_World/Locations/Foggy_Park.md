# 공원 상세 설정

## 1. 기본 정보

- **공식 명칭**: 공원
- **영문명**: Foggy Park
- **위험도**: 비교적 안전
- **중심성**: 중심부 근처
- **세력 영향**: 방랑자
- **ConfigureBook 원고**: `ConfigureBook/Place/Park.md`
- **Import 상태**: Reviewing

## 2. 역할

공원은 방랑자의 은신처이자 암상인 접촉 가능 지역이다. 위험도는 낮지만 안개, 호수, 숲과 정자가 만드는 시야 제한 때문에 추적, 은닉, 실종 이벤트에 적합하다.

## 3. 세부 구역

- 산책로
- 호수 근처
- 정자

## 4. 이벤트 제작 방향

- 방랑자와의 거래 또는 교섭
- 금화를 사용하는 암상인 이벤트
- 안개 속 미행과 실종
- 호수 주변 이상현상
- 낮은 위험도와 불안한 분위기의 대비

## 5. 원고 반영 메모

ConfigureBook 원고의 안개, 숲, 호수 묘사와 암상인 시드는 후속 Import 패스에서 세부화한다.
