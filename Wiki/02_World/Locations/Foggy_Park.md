# 공원 상세 설정

## 1. 기본 정보

- **공식 명칭**: 공원
- **영문명**: Foggy Park
- **위험도**: 비교적 안전
- **중심성**: 중심부 근처
- **세력 영향**: 방랑자
- **ConfigureBook 원고**: `ConfigureBook/Place/Park.md`
- **Import 상태**: Imported Candidates Added

## 2. 역할

공원은 방랑자의 은신처이자 암상인 접촉 가능 지역이다. 위험도는 낮지만 안개, 호수, 숲과 정자가 만드는 시야 제한 때문에 추적, 은닉, 실종 이벤트에 적합하다.

공원의 안개는 단순한 기상 현상이 아니라 소리를 흡수하고 시야를 왜곡하는 이상현상 후보로 사용한다.

## 3. Import 기준

이 문서의 이상현상은 항상 발생하는 고정 사건이 아니라 이벤트 제작 시 사용할 수 있는 후보 목록이다.

자원 획득 후보와 등장 가능 괴물·세력 후보는 현재 정본 DB와 대조가 끝나지 않았으므로 본문에 확정 등록하지 않는다.

- **자원 후보**: TODO — Resource System 또는 Item DB 정리 후 등록
- **괴물/세력 후보**: TODO — Monster DB 및 Entity 문서와 대조 후 등록

## 4. 세부 구역

### 4.1 외곽 및 산책로

#### 공원 정문 및 매표소

- **공간 성격**: 공원 진입 통제와 거래 전초지
- **상호작용 후보**: 통행료 지불, 입구 근처 매복, 정보 교환
- **이상현상 후보**: 매표소 티켓에 읽을 때마다 다른 죽음의 날짜가 찍혀 나옴

#### 조깅 트랙 및 숲길

- **공간 성격**: 은밀한 이동과 길 잃음 발생지
- **상호작용 후보**: 은밀한 이동, 숲길 정찰, 약초·소형 물자 후보 조사
- **이상현상 후보**: 안개가 짙어질 때 트랙을 따라 뛰는 정체 모를 발자국 소리가 들리고, 따라가면 길을 잃음

### 4.2 방랑자의 거처

#### 중앙 광장 및 파빌리온

- **공간 성격**: 방랑자 캠프와 대규모 거래 중심
- **상호작용 후보**: 물자 거래 후보, 방랑자 퀘스트 수주, 요리와 휴식 후보
- **이상현상 후보**: 드럼통 화로의 불꽃이 보라색으로 변하고 죽은 이들의 얼굴이 나타남

#### 공중 화장실

- **공간 성격**: 임시 창고, 은신처, 뒷거래 목격지
- **상호작용 후보**: 물품 은닉, 도청, 뒷거래 목격
- **이상현상 후보**: 닫힌 변기 칸 안에서 누군가 숫자를 세는 소리가 계속 들림

#### 어린이 놀이터

- **공간 성격**: 방랑자 생활권과 잃어버린 아이들에 대한 공포
- **상호작용 후보**: 고철 수집 후보, 함정 설치 연습, 방랑자 아이들과 상호작용
- **이상현상 후보**: 모래사장에 발을 들이면 차가운 손의 감촉이 느껴지고, 확인하면 낡은 인형 팔만 나옴

### 4.3 위험 경계 및 특수 구역

#### 안개 낀 호수

- **공간 성격**: 공원 안에서 가장 위험한 경계 구역
- **상호작용 후보**: 낚시 후보, 희귀 자원 후보 탐색, 위험한 도박성 탐색
- **이상현상 후보**: 호수를 오래 바라보면 수면 아래에서 자신과 똑같은 형체가 올라오려 함

#### 식물원 및 온실

- **공간 성격**: 변이 식물과 죄책감 이상현상 후보지
- **상호작용 후보**: 식물 조사, 독성 중화 후보, 은밀한 대화 이벤트
- **이상현상 후보**: 온실 꽃들이 플레이어의 말을 흉내 내고, 침식이 높을 경우 죄책감을 속삭임

## 5. 이벤트 제작 방향

- 방랑자와의 거래 또는 교섭
- 금화를 사용하는 암상인 이벤트
- 안개 속 미행과 실종
- 호수 주변 이상현상
- 낮은 위험도와 불안한 분위기의 대비

## 6. Import 메모
