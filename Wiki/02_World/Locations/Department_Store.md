# 백화점 상세 설정

## 1. 기본 정보

- **공식 명칭**: 백화점
- **영문명**: Department Store
- **위험도**: 접근금지
- **중심성**: 외곽 심부
- **세력 영향**: 광신도
- **ConfigureBook 원고**: `ConfigureBook/Place/Department_Store.md`
- **Import 상태**: Reviewing

## 2. 역할

백화점은 거대한 상업 폐허이자 괴생명체들의 둥지다. 광신도는 이곳을 위대한 시험으로 여기며, 생존자가 들어서는 순간 막대한 자원 보상과 거의 확실한 파멸 위험이 함께 열린다.

## 3. 세부 구역

- 쇼핑 플로어
- 식품관
- 의식 구역

## 4. 이벤트 제작 방향

- 대량 자원 탐색
- 괴물 둥지 침입
- 광신도의 시험
- 미로형 내부 구조와 실종
- 소비 공간이 제단으로 변질된 공포

## 5. 원고 반영 메모

ConfigureBook 원고의 층별 구조, 매장 폐허와 의식 구역 묘사는 후속 Import 패스에서 세분화한다.
