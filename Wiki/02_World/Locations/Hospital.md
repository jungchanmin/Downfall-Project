# 병원 상세 설정

## 1. 기본 정보

- **공식 명칭**: 병원
- **영문명**: Hospital
- **위험도**: 매우 위험
- **중심성**: 외곽
- **세력 영향**: 자경단
- **ConfigureBook 원고**: `ConfigureBook/Place/Hospital.md`
- **Import 상태**: Reviewing

## 2. 역할

병원은 귀중한 의약품과 의료 장비가 남아 있는 고위험 지역이다. 자경단 주둔으로 통제가 이루어지지만, 병원 내부의 오염과 변이 생물 때문에 탐색과 치료가 항상 위험을 동반한다.

## 3. 세부 구역

- 외래진료실
- 입원실
- 약국
- 수술실
- 연구실

## 4. 이벤트 제작 방향

- 의약품 확보와 치료 선택
- 자경단 통제와 의료 지원 거래
- 변이 생물 격리 실패
- 수술실과 연구실의 윤리적 공포
- 감염, 오염, 생존자 부상 이벤트

## 5. 원고 반영 메모

ConfigureBook 원고의 병동, 연구실, 의료 장비 묘사는 후속 Import 패스에서 상세 반영한다.
