# 👹 DOWNFALL 괴물 데이터베이스 (v0.1 — 분류 체계)

> 이 문서는 **분류 체계와 연계 훅**까지만 확정한다.
> 개체별 스탯 블록(SECTION 3)은 후속 단계에서 채운다.

---

## SECTION 0 — 분류 2축 모델

괴물은 두 개의 **직교(orthogonal)** 축으로 표현된다.
개체 하나는 `(위협 등급 × 형태)` 좌표 하나로 정의된다.

### 0-1. 축 1 — 위협/지속성 (지도에 남는가)

| 위협 등급 | 정의 |
|---|---|
| **일반괴물 (Instant)** | 이벤트·탐색·세력교류·지역특성으로 **일시 생성**. 즉시 전투. 도주·격퇴 성공 시 **소멸**. 지도에 남지 않음. |
| **네임드괴물 (Persistent)** | 매일 **새벽 페이즈** 출현, 마을 지역에 **상주**. 처치될 때까지 영향 지속. 지도에 남음. |

네임드 하위 분류:

```
표준형        : 강하지만 통상적인 네임드
돌연변이형     : 변이로 특수 능력·기형을 획득한 네임드
멸망의 괴물    : 최상위. 특정 개체는 일 단위 지속 재앙을 유발
```

> ⚠️ 명칭: 네임드 하위 "표준형"은 최상위 "일반괴물"과 충돌을 피하기 위해 채택한 명칭이다.
> ("일반형" 사용 금지)

### 0-2. 축 2 — 형태 (Morphology) · 일반·네임드 공용

| 형태 | 정의 |
|---|---|
| **인간형** | 적대적 생존자. 무력으로 전투 인카운터 진입 가능한 인간 존재. (SECTION 1-A) |
| **① 동물형** | 특정 동물의 혼합·거대화·지능화. 네임드 대비 훨씬 약함(일반 기준). |
| **② 곤충형** | 동물형에 준함. 곤충 기반. |
| **③ 심령체형** | 유령·귀신에 가까운 형태. |
| **④ 유사인간형** | 좀비·구울·도플갱어 등 인간을 모방·왜곡한 형태. |
| **⑤ 이상현상형** | 사물·괴이 현상이 괴물처럼 움직임 (그림자, 피웅덩이, 목을 조르는 그림). |
| **⑥ 이계급괴이형** | 거대 괴물(심해 괴수·거인)이 본체 전부가 아닌 **일부만 드러내** 강림. |
| **⑦ 촉수·괴물형** | 슬라임·촉수 등의 형태. |

### 0-3. 식별자 규약 (신규 태그 카테고리 — 승인 완료)

```
#Monster_{형태}_{이름}
예시:
  #Monster_Animal_GiantCrow        (일반 × 동물형)
  #Monster_Humanoid_Looter         (일반 × 인간형)
  #Monster_Abyssal_DrownedArm      (네임드-돌연변이 × 이계급괴이형)
```

> `#Monster_` 는 기존 스키마(`#Stat_`/`#Trait_`/`#Loc_`/`#Rel_`/`#Flag_Memory_`)에
> 추가된 신규 카테고리이며 디렉터 승인을 받았다.

## SECTION 1 — 일반괴물 (Instant)

### 1-0. 공통 규칙

```
생성 조건: 지역 탐색/탐험 · 세력교류 · 이벤트 결과 · 지역 특성
전투 진입: 즉시 (인스턴트)
종료 조건: 도주 성공 또는 격퇴 성공 → 즉시 소멸
지도 잔존: 없음
```

### 1-A. 인간형 일반괴물 (적대적 생존자)

무력으로 전투에 진입 가능한 인간 존재.

```
무소속:
  강도 · 무기를 든 생존자 무리 · 약탈자 · 괴한 · 납치범
세력 소속(말단 조직원):
  부랑자(방랑자 세력) · 자경단원 · 악마조직원 · 광신도
```

#### 세력 회피 훅 (Faction Resolution)

> 인간형 일반괴물이 *세력 소속*인 경우, 전투 외에 **세력 회피** 경로가 열린다.
> 이것이 적대적 생존자를 "강도"가 아니라 "방랑자/자경단원"으로 구분하는 이유다.

```
- 세력 영향 구역에서는 해당 세력 소속 적대 생존자의 등장 확률이 상승한다.
- faction 태그를 가진 인간형 일반괴물은,
  전투 대신 [세력호감도] 소비/판정으로 회피·해결 가능하다.
  → 유저 만족 장치: 쌓아둔 세력 호감도가 전투를 우회하는 보상으로 작동.
- Monster DB는 전투 스탯·정보만 저장한다.
  세력 소속은 faction 태그로 FACTION_DB와 연결한다. (데이터 복제 없음)
```

### 1-B. 비인간형 일반괴물

SECTION 0-2 의 7개 형태(①~⑦)를 따른다.
**일반 등급의 동물형·곤충형은 네임드 대비 현저히 약하게** 설정한다.

## SECTION 2 — 네임드괴물 (Persistent)

### 2-0. 공통 규칙

```
출현 조건: 매일 새벽 페이즈
출현 위치: 마을 지역 (지도 상주)
영향 지속: 처치될 때까지
전투 진입: 등장 시 즉시 전투 아님 (SECTION 6 참조)
특성:      매우 강함 · 높은 지능 · 개체 고유 특색 보유
```

### 2-A. 표준형
강하지만 통상적인 네임드. 형태 축 ①~⑦ + 인간형 적용 가능.

### 2-B. 돌연변이형
변이로 특수 능력·기형을 획득. 표준형보다 까다로움.

### 2-C. 멸망의 괴물
최상위 위협. **특정 개체는 일 단위 지속 재앙을 유발**하여 생존자 전체를 위협한다.

### 2-D. 날짜 타입 출현 훅 (보류)

> ⚠️ 날짜 타입(고요의 날 / 전조의 날 / 멸망의 날)은 현재 위키에 정의되지 않았다.
> 날짜별 세계 이벤트 정의가 아직 없어 **보류** 상태다.
> 향후 `MECH_Time_System` 에 날짜 타입을 편입한 뒤 본 훅을 수치 연동한다.

```
[미정] 고요의 날  → 출현 가능 네임드 풀: ...
[미정] 전조의 날  → 출현 가능 네임드 풀: ...
[미정] 멸망의 날  → 출현 가능 네임드 풀: (멸망의 괴물 포함)
```

## SECTION 3 — 개체 데이터 양식 (DRAFT — 다음 단계 확정)

> 아래는 제안 양식이다. 필드 확정은 후속 단계로 미룬다.

```yaml
monster:
  id: "#Monster_{형태}_{이름}"
  threat: 일반 | 네임드-표준형 | 네임드-돌연변이 | 네임드-멸망
  morphology: 인간형 | 동물형 | 곤충형 | 심령체형 | 유사인간형 | 이상현상형 | 이계급괴이형 | 촉수괴물형
  spawn:
    condition: "[생성/출현 조건]"
    location: "[Loc 또는 지역 특성]"
  combat:
    repel_dc: N        # 격퇴 판정 난이도
    flee_dc: N         # 도주 판정 난이도
    threat_penalty: "[패배 시 패널티 — Reward Appraisal T 사다리 연동]"
  weakness:
    clue_gated: true | false   # 단서로 약점 해제 가능 여부
  reward:
    on_defeat: "[격퇴 성공 보상 — Reward Appraisal R 사다리 연동]"
  faction: "[인간형 한정. 세력 회피 훅 연결. 없으면 생략]"
```

## SECTION 6 — 보상 책정기 연계 ★

`MECH_Event_Reward_Appraisal` SECTION 8 의 T5 훅 "괴물 출현"과 연결된다.
**패널티 강도에 따라 스폰 대상이 갈리며, 전투 진입 방식이 다르다.**

| 패널티 강도 | 스폰 대상 | 전투 진입 방식 |
|---|---|---|
| **낮음** | 일반괴물 | **즉시 전투 진입.** 도주·격퇴 성공 시 그대로 종료(소멸). |
| **높음** | 네임드괴물 | **등장만 한다.** 즉시 전투 아님. 지도에 상주 추가되어 이후 위협으로 작동. |

```
핵심 구분:
  일반괴물 = 인스턴트. 그 자리에서 싸우고 끝난다.
  네임드   = 상주. "나타났다"는 사실 자체가 결과이며, 전투는 이후 별도로 발생한다.
```

> 따라서 보상 책정기가 T5 에서 네임드를 스폰하는 것은
> "즉시 패배 위협"이 아니라 "지속 위협의 등장"이라는 더 무거운 처벌이다.
