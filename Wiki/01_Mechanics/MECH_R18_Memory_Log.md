# 📓 DOWNFALL R18 기록·변질 시스템 (v2.0)

> R18은 **제거 가능한 토글 모듈**. SFW 빌드에서 본 시스템 전체 비활성. 성인(18+) 캐릭터 한정.
> 다운폴 메모리 플래그(`Flag_Memory_*`)의 카운터 확장으로 구현된다.

---

## SECTION 0 — 설계 원칙

1. **2층 곱연산 구조**: 대분류 = *변질 방향*(거시), 소분류 = *부위 민감도*(미시 약점). 둘이 곱연산.
2. **변질의 방향성**: 축마다 "인간에서 벗어나는 방향"이 다르다. 어느 축이 높은지가 그 NPC가 *무엇이 되어가는지*를 규정.
3. **3단 계단**: 기록(서사) → 숙련(전술) → 업적(전략·양날). 업적은 진행 규칙을 바꿀 정도이되 반드시 양날.
4. **수동 기본 + 제한적 능동**: 기본 진척은 괴물 주도(수동). 호러를 위해 통제 불가가 기본. 능동 가속은 굴복 페이즈·양날 커맨드로만.
5. **상호보완**: R18 기벽 DB와 충돌 아닌 보완 — 해금→기벽 부여, 기록↔기벽 정합성 제약, 특성→기벽 완화.

---

## SECTION 1 — 대분류: 변질 6축 (Corruption Axes)

> 행위가 아니라 **변질의 방향**으로 묶는다. 개별 커맨드(끌어안기·주무르기 등)는 해당 축으로 흘러든다.

| 축 | 정의 | 방향 | 우선 구현 |
|---|---|---|---|
| **① 접촉 침윤 (Contact)** | 살이 닿고 길들여짐 | 겉 → 거부감 소실 | 골격 |
| **② 침범 수용 (Penetration)** | 몸 안으로 받아들임 | 경계 → 수용 | 골격 |
| **③ 체액 동화 (Fluid)** | 마시고 흡수해 몸이 변함 | 거부 → 의존 | ★ 우선 |
| **④ 변이 각인 (Mutation)** | 신체 구조 자체가 개조됨 | 인간 → 비인간 신체 | ★ 우선 |
| **⑤ 인지 개변 (Cognition)** | 정신·상식이 인간에서 벗어남 | 상식 → 왜곡 | ★ 우선 |
| **⑥ 결착 함락 (Subjugation)** | 절정·굴복으로 무너짐 | 저항 → 함락 | 골격 |

```
축 카운터 키: Log_R18_{축}  (예: Log_R18_Fluid)
같은 침식도라도 어느 축이 높은지에 따라 다른 종류의 변질체가 된다.
③체액·④변이·⑤인지 = 내부·정신·구조 변화 → "인간에서 벗어남"을 가장 강하게 표현. 우선 구현.
①접촉·②침범·⑥함락 = 골격만 예약, 추후 확충.
```

### 1-1. 축별 능동/수동 비중
```
겉 축(①접촉·②침범)   : 거의 수동 (당해서 쌓임). 호러적 무력감.
내부 축(③체액·④변이)  : 능동 가속 가능 (스스로 받아들이는 선택 — 굴복 페이즈 흡수형 등).
정신 축(⑤인지)        : 혼합 (당함 + 상식 개변 선택).
⑥함락                : 굴복 페이즈 결착으로만.
```

## SECTION 2 — 소분류: 신체부위 민감도 (경험치형)

> 대분류 축 아래, **부위별 누적이 경험치처럼 쌓여 민감도 레벨**이 오른다. 개발된 부위가 약점이자 무기.

### 2-1. 구조
```
부위 카운터 = Log_R18_{부위}_{행위}  (기존 v1 카운터 재해석)
부위 = 입 / 가슴 / 하복부 / 후부 / 전신
행위 = 괴롭혀짐 / 절정당함 / 개발당함 / 가함

민감도 레벨: 부위 카운터 임계마다 Lv0 → Lv1 → Lv2 ...
효과(곱연산):
  - 같은 부위 공격 시 해당 변질 축 진척 배율 ↑
  - 그 부위 한정 성감 누적 배율 ↑
  - 레벨 임계 도달 시 감도 개발 계열 R18 기벽 해금(MECH_Quirk_R18_DB 연동)
```

### 2-2. 전략적 양면성
```
약점화: 개발된 부위는 같은 공격에 점점 취약 → 분산 방어의 동기.
무기화: 굴복 페이즈에서 개발된 부위를 역으로 활용(흡수형·농락형 위력↑).
→ 플레이어는 "개발을 막을 것인가 / 감수하고 무기로 쓸 것인가"를 선택.
```

### 2-3. 곱연산 예시
```
진척 증가량 = 공격 기본치 × 축 보정 × (1 + 부위 민감도 레벨 보정)
예) 가슴 접촉(기본 2) × 가슴 Lv2(+0.5) → 접촉 축 진척 3, 가슴 성감 배율 추가
```

## SECTION 3 — 3단 계단 효과 (축별 공통 틀)

| 단계 | 성격 | 내용 |
|---|---|---|
| **1단 기록(Record)** | 서사 | 카운터만 누적. 효과 없음. 상태창에 변질 텍스트 로그 표시("…거부감을 느끼지 않는다") |
| **2단 숙련(Proficiency)** | 전술 | 임계 도달 시 전투 내 점진 효과(해당 축 관련 저항/배율/대응) |
| **3단 업적(Milestone)** | 전략·양날 | 높은 임계. 진행 규칙을 바꿀 변화 + 영구 디버프 동반 |

> 1단은 공짜(서사), 2단은 점진 편의, 3단만 게임체인저이자 양날. 강도를 계단으로 나눠 밸런스 유지.

## SECTION 4 — 우선 구현 3축 단계표

### 4-1. ③ 체액 동화 (Fluid)
```
1단 기록: "비릿한 것이 입에 닿아도 더는 토하지 않는다."
2단 숙련(임계 3): 거부 반응 약화 — 괴물 체액 피격 시 스트레스 누적 감소
3단 업적(임계 5): 동화 — 체액 섭취 시 일시 능력 버프 (#Trait_Semen_Dependency 해금, 양날)
3단 업적(임계 8): 의존 — 체액 미섭취 시 컨디션 페널티. 식량 대신 체액을 갈구
                 → 생존 자원 루프 변경(식량 의존↓·괴물 의존↑). [진행 규칙 수정]
```

### 4-2. ④ 변이 각인 (Mutation)
```
1단 기록: "살갗 아래에서 낯선 무게가 자라는 감각."
2단 숙련(임계 3): 변이 부위 둔감화 — 해당 부위 통증 페널티 경감
3단 업적(임계 5): 신체 변이 계열 R18 기벽 1개 영구 부여(예: 살점 융기). [완화 불가 — 영구]
3단 업적(임계 8): 비인간 신체 — 특정 행동 판정 보정 변화 + 외형 변질로 세력 교류 페널티
                 [진행 규칙 수정: 일부 NPC 상호작용 차단]
```

### 4-3. ⑤ 인지 개변 (Cognition)
```
1단 기록: "이상하다고 느껴야 할 일이, 더는 이상하지 않다."
2단 숙련(임계 3): 공포 저항 — R18 전투기벽 발현 임계 상승
3단 업적(임계 5): 상식 개변 계열 R18 기벽 해금(예: 가학 복종의 정당성, 양날)
3단 업적(임계 8): 왜곡 고착 — 특정 위험 선택지를 '정상'으로 인지 → 자동/강제 선택 분기
                 [진행 규칙 수정: 리더 통제 일부 상실]
```

> ①접촉·②침범·⑥함락은 동일 틀로 추후 확충(현재 골격만).

## SECTION 5 — 진척 발생 (수동/능동)

```
수동(기본): 괴물의 R18 커맨드 적중 → 해당 축 + 피격 부위 카운터 증가. 통제 불가(호러).
능동(제한):
  - 굴복 페이즈 흡수형 기술: 체액/변이 축 진척을 능동 가속 (회복과 맞바꿈, MECH_Combat_System SECTION 11)
  - 양날 R18 기벽(역이용 등): 침식·진척을 자원으로 소모/상승
부위 민감도: 그 부위가 관여한 공격마다 부위 카운터 증가 → 곱연산으로 축 진척 가속.
```

---

## SECTION 6 — 완화 (역행 루트)

```
조건: 특정 해금 특성 보유 + 반대 방향 행위 기록 누적('가함' 기록으로 '당함' 고착 상쇄)
효과: 단계형·지속형 R18 기벽(감도·상식 개변) 단계 1 하향 또는 일시 억제. 축 진척 일부 감소.
제약: 영구 변이(④의 영구 부여분)는 완화 불가. 완전 타락을 막는 여지만 제공.
```

## SECTION 7 — R18 기벽 DB 상호보완

| 관계 | 규칙 |
|---|---|
| 해금 → 부여 | 축/부위 임계 도달 시 `grants_quirk`로 R18 기벽 부여 |
| 기록 ↔ 정합성 | 해당 부위/축 기록 0이면 그 계열 기벽 부여 차단 |
| 기벽 → 완화 | 단계·지속형 기벽은 SECTION 6 완화 대상. 영구 변이 제외 |
| 기벽 → 커맨드 강화 | R18 기벽 보유 시 굴복 페이즈 전용 기술 위력↑ (Combat SECTION 11) |

## SECTION 8 — 구현 데이터 구조

```yaml
R18MemoryLog:                  # NPC별
  axis_counters: { Fluid: int, Mutation: int, Cognition: int, Contact: int, Penetration: int, Subjugation: int }
  part_exp: { "가슴_괴롭혀짐": int, "입_개발당함": int, ... }   # 부위 경험치
  part_level: { 가슴: int, 입: int, ... }                      # 파생 민감도 레벨
  unlocked_milestones: [id]

on_r18_hit(npc, axis, part, act, base):
    npc.axis_counters[axis] += base * axis_mod * (1 + sens_bonus(npc.part_level[part]))
    npc.part_exp[f"{part}_{act}"] += 1
    recompute_part_level(npc, part)         # 경험치 → 레벨
    check_milestones(npc, axis)             # 3단 임계 체크 → 효과/기벽(정합성 검사 후)
```
