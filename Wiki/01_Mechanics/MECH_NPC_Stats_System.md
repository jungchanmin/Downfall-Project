# ⚙️ NPC 생존자 스탯 시스템

---

## SECTION 1 — 스탯 체계 개요

NPC 생존자의 스탯은 4개 계열로 구분된다.

| 계열 | 종류 | 특성 |
|---|---|---|
| **기본 스탯** | 체력 / 정신력 / 스트레스 / 허기 / 컨디션 / 피로도 | 실시간으로 변동. 생존의 직접 지표 |
| **능력 스탯** | 전투 / 탐색 / 대화 / 간호 / 제작 / 기술 / 매력 / 지능 | 변동 어려움. NPC 간 개인차 지표 |
| **감정/사회 스탯** | 불만도 / 적응도 / 상태이상 / 욕구불만 / 친밀도 | 서술적 묘사로 제공. 관계·행동 결정 |
| **성격 스탯** | 신경성 / 인내심 / 외향성 / 공격성 / 성욕 / 양심 | 고정값. NPC의 원천적 성격 |

---

## SECTION 2 — 기본 스탯

### 2-1. 체력 (HP)

```
범위: 0 ~ 100
초기값: NPC별 상이
상한값: NPC별 상이 (일반적으로 초과 불가)
```

**역할**: 생존자의 신체적·육체적 한계치.

**상호작용**:
- 허기가 0에 달하면 체력 감소
- 전투·부상·혹독한 환경에서 감소
- 간호 스탯 판정 성공 시 회복 가능

**임계값**:
```
체력 = 0 → 생존자 사망
체력 <= 20 → 부상 상태 (능력 스탯 보정 적용)
```

---

### 2-2. 정신력 (Sanity)

```
범위: 0 ~ 100
초기값: NPC별 상이
상한값: NPC별 상이
```

**역할**: 생존자의 정신적·심리적 상태.

**상호작용**:
- **스트레스 최대치 도달 시** 정신력 일정량 감소
- F5_Cosmic 계열 이벤트에서 직접 감소 가능
- 극도의 공포·충격 이벤트에서 직접 감소 가능

**임계값**:
```
정신력 = 0   → 생존자 완전 발광. 사망 처리.
정신력 <= 20 → 기벽 발현 조건 충족 가능
정신력 <= 40 → 고위험 정신 상태. 돌발 이벤트 발생 확률 증가
```

---

### 2-3. 스트레스 (Stress / Mental)

```
범위: 0 ~ 10
```

**역할**: 일상적 압박·피로·공포의 누적 지표. 대부분의 이벤트가 정신력 대신 스트레스에 영향을 준다.

**상호작용**:
```
스트레스 최대치(10) 도달 시:
  → 정신력 일정량 감소 (NPC별 상이)
  → 스트레스 절반(5)으로 감소
```

**이벤트 변동 기준**:
```
심각한 공포 이벤트:  +2 ~ +3
중간 위협 이벤트:    +1 ~ +2
긍정적 이벤트:       -1 ~ -2
충분한 휴식:         -1
```

**1회 변동 상한**: ±3

---

### 2-4. 허기 (Hunger)

```
범위: 0 ~ 10
한 끼 식량 섭취량: 1
```

**역할**: 굶주림 지표. 식량 소비로 해소.

**상호작용**:
```
허기 = 0 → 체력 감소 시작
허기 증가 → 체력에 영향 없음 (단방향)
```

**제한 사항**:
- 한 끼에 식량 1개만 섭취 가능
- 식량을 대량 제공해도 한 번에 대량 회복 불가
- 페이즈당 1회 식사 기회

---

### 2-5. 컨디션 (Condition)

```
범위: -5 ~ +5
기준값: 0
```

**역할**: 능력 스탯 전체에 긍정/부정 보정을 주는 지표.

**상호작용**:
- 친밀도 이벤트 성공 시 증가
- 부정적 이벤트·갈등 시 감소
- 긍정적 대화·교류·휴식으로 증가

**능력 스탯 보정**:
```
컨디션 +3 ~ +5: 능력 스탯 +1 보정
컨디션 0 ~ +2:  보정 없음
컨디션 -1 ~ -2: 능력 스탯 -1 보정
컨디션 -3 ~ -5: 능력 스탯 -2 보정
```

---

### 2-6. 피로도 (Fatigue)

```
범위: 0 ~ 10
```

**역할**: 활동 누적 피로. 높을수록 능력 스탯에 부정적 보정.

**증가 조건**:
```
불침번 수행:  +1
외출 탐색:    +1 ~ +2
전투 참여:    +2 ~ +3
```

**해소 조건**:
```
충분한 수면:  -2 ~ -3
간호 받기:    -1 ~ -2
긍정적 대화:  -1
```

**능력 스탯 보정**:
```
피로도 0 ~ 3:  보정 없음
피로도 4 ~ 6:  능력 스탯 -1 보정
피로도 7 ~ 9:  능력 스탯 -2 보정
피로도 = 10:   능력 스탯 -3 보정 + 스트레스 +1/페이즈
```

---

## SECTION 3 — 능력 스탯

```
범위: 0 ~ 10 (NPC별 초기값·상한값 상이)
상승 조건: 훈련·경험 누적 (어지간하면 상승 어려움)
```

### 3-1. 전투 (Combat)

괴물·적대 생존자와의 전투, 위압, 물리적 해결 상황.

**특화 NPC 예시**:
- 하인츠 폰 파펜 (군인 출신)
- 레이첼 로페즈 (국가대표 펜싱)
- 아그네스 비시드 (투포환 세계기록 보유자)
- 안토니오 바르바라 (마피아 출신)

---

### 3-2. 탐색 (Search)

정보 획득, 관찰, 이상 징후 발견, 유용한 자원 탐색, 괴물 약점 파악, 위험 감지.

**특화 NPC 예시**:
- 헬레나 비노슈 (국가비밀요원)
- 올리버 호손 (사설탐정)
- 제인 캘러핸 (경찰)

---

### 3-3. 대화 (Dialogue)

생존자 간 대화·설득·친밀도 교류, 협상, 지성 있는 괴물과의 의사소통 시도.

---

### 3-4. 간호 (Care)

의술, 상처 파악, 의료 지식, 응급 처치, 심리 상담, 상처 처치.
부상자 체력 회복 및 스트레스 관리에 사용.

---

### 3-5. 제작 (Craft)

장비 제작, 수리, 시설 강화, 물건 가공, 용접, 강화.
아지트 방어 및 자원 활용에 직결.

---

### 3-6. 기술 (Knowledge)

종합적 지식, 상황 대처, 능동적 대응, 전문 지식.
다양한 이벤트 해결에 범용적으로 기여.

---

### 3-7. 매력 (Charm)

매력, 미모, 호감, 유혹, 미인계.
NPC 관계 및 세력 교류에서 유리하게 작용.

---

### 3-8. 지능 (Intelligence)

뛰어난 사고, 독파, 지식 습득, 지능적 계산, 영리함.
복잡한 문제 해결 및 전략 수립에 영향.

---

## SECTION 4 — 감정/사회 스탯

서술적 묘사로 제공. 수치는 내부 계산용.

### 4-1. 불만도 (Dissatisfaction)

```
범위: 0 ~ 10
```

조직·리더에 대한 불만 누적.

```
불만도 >= 7: 명령 거부 가능성 발생
불만도 = 10: 반항·이탈 이벤트 강제 발동
```

---

### 4-2. 적응도 (Adaptation)

```
범위: 0 ~ 10
```

상황·조직·생존 환경에 대한 순응도.

```
적응도 <= 3: 스트레스 증가 보정 +1
             돌발 이벤트 발생 확률 증가
```

---

### 4-3. 상태이상 (Status Abnormality)

정신적/신체적 약화 상태. 서술적으로 관리.

**예시**:
- 면역력 감퇴
- 수면 장애
- 공황 발작
- 기벽 발현 (정신력 임계값 연동)

---

### 4-4. 욕구불만 (Frustration)

```
범위: 0 ~ 10
```

육체적·정신적·성적·취미 욕구 미충족 누적.

```
욕구불만 >= 7: 비이성적 행동 이벤트 발생 가능
욕구불만 = 10: 돌발 행동 강제 발동
```

---

### 4-5. 친밀도 (Intimacy)

```
범위: 0 ~ 10
NPC별 개별 계산 (리더↔NPC / NPC↔NPC)
```

관계 발전·갈등 해결의 핵심 지표.

**분기 임계값 (이벤트 설계 기준)**:
```
친밀도 0 ~ 1:  초면·경계 관계
친밀도 2 ~ 4:  신뢰 형성 단계
친밀도 5 ~ 7:  우호적 관계
친밀도 8 ~ 10: 강한 유대·의존 관계
```

**이벤트 분기 기준값 (현행)**:
```
친밀도 낮음: intimacy < 2
친밀도 높음: intimacy >= 2
```

> ⚠️ 임시 기준값. 전체 밸런스 확정 후 조정 예정.

---

## SECTION 5 — 성격 스탯

**고정값. 변경 불가.** NPC의 원천적 성격.
이벤트 판정 modifier 및 행동 패턴 결정에 사용.

```
범위: 0 ~ 10 (고정)
```

### 5-1. 신경성 (Neuroticism)

예민함·민감함·눈치·직감.

```
신경성 높음(7~10): 돌발 이벤트·공포에 민감 반응
                   스트레스 증가 modifier +1
신경성 낮음(0~3):  충격에 둔감. 공포 이벤트 DC 보정 +1
```

---

### 5-2. 인내심 (Patience)

참을성·인내력·의지.

```
인내심 높음(7~10): 스트레스 증가 저항
                   스트레스 변동 -1 보정
인내심 낮음(0~3):  불만도 증가 속도 빠름
                   장기 생존 이벤트에서 불리
```

---

### 5-3. 외향성 (Extraversion)

사교성·친절함·활동성.

```
외향성 높음(7~10): 대화 스탯 판정 +1 보정
                   친밀도 증가 속도 빠름
외향성 낮음(0~3):  단독 이벤트 스트레스 저항 +1
                   대화 판정 -1 보정
```

---

### 5-4. 공격성 (Aggression)

과감함·공격성·성급함.

```
공격성 높음(7~10): 전투 스탯 판정 +1 보정
                   불만도 증가 속도 빠름
공격성 낮음(0~3):  신중한 행동. 탐색 판정 +1 보정
```

---

### 5-5. 성욕 (Libido)

이성·욕망에 이끌리는 정도.

```
성욕 높음(7~10): 욕구불만 증가 속도 빠름
                 특정 R18 이벤트 분기 진입 조건
성욕 낮음(0~3):  욕구불만 증가 저항
```

---

### 5-6. 양심 (Conscience)

개인의 도덕적 가치.

```
양심 높음(7~10): 비윤리적 명령 수행 시 불만도 증가
                 윤리적 선택 이벤트에서 특수 분기 진입
양심 낮음(0~3):  비윤리적 명령 수행 시 불만도 영향 없음
                 잔혹한 선택 이벤트에서 패널티 없음
```

## SECTION 6 — 이벤트 판정 체계

### 6-1. 기본 판정 구조

```yaml
stat_check:
  stat: [검사 능력 스탯]
  dc: [난이도]
  modifiers:
    - condition: [조건]
      bonus: +N | -N
  result:
    success: dc 이하
    fail: dc 초과
```

> ⚠️ 주사위 체계 미확정. DC는 현재 절대값 기준.
> 추후 주사위 시스템 도입 시 재조정 예정.

---

### 6-2. DC 기준표

```
DC 8  — 매우 쉬움. 평균적 NPC라면 대부분 통과.
DC 10 — 보통. 해당 스탯이 5 이상이면 안정적 통과.
DC 12 — 어려움. 특화 NPC 또는 modifier 없으면 실패 가능.
DC 13 — 높음. F5_Cosmic 계열 기본값.
DC 15 — 매우 어려움. 최고 특화 NPC도 실패 가능.
DC 18 — 극한. 거의 실패. 특수 상황 전용.
```

---

### 6-3. Modifier 적용 기준

```
성격 스탯 기반:    ±1 ~ ±2
기본 스탯 기반:    ±1 ~ ±2
플래그 기반:       ±2 ~ ±3
컨디션/피로도:     ±1 ~ ±2 (SECTION 2-5·2-6 기준)
```

**1회 modifier 누적 상한**: ±5
(과도한 누적으로 판정이 무의미해지는 것을 방지)

---

### 6-4. 스탯 변동 상한 (1회 이벤트 기준)

```
기본 스탯 변동: ±3 이내
능력 스탯 변동: ±1 이내 (훈련·경험 이벤트 한정)
친밀도 변동:    ±1 이내
침식도 변동:    ±3 이내
```

---

## SECTION 7 — 침식도 (Erosion)

> ⚠️ 침식도는 기본 스탯·감정 스탯과 별개로 관리되는
> **R18 트랙 전용 특수 스탯**이다.

```
범위: 0 ~ 100
```

**역할**: 타락·붕괴 진척도. NPC의 심리적 방어선이 무너지는 정도.

**분기 임계값 (이벤트 설계 기준)**:
```
침식도 0 ~ 29:   방어선 유지. 절제된 반응.
침식도 30 ~ 59:  방어선 균열. 타락 충동 발현 시작.
침식도 60 ~ 89:  방어선 붕괴. 강박과 충동 충돌.
침식도 90 ~ 100: 완전 타락. 엘리트 강박 소멸.
```

**이벤트 분기 기준값 (현행)**:
```
침식도 낮음: erosion < 30
침식도 높음: erosion >= 30
```

> ⚠️ 임시 기준값. 전체 밸런스 확정 후 조정 예정.

**증가 조건**:
```
R18 이벤트 ch1 친밀도 높음 결과: +2 ~ +3
R18 이벤트 ch2 친밀도 높음 결과: +1 ~ +2
```

## SECTION 8 — 자원 체계 참조

> 자원 체계는 별도 문서 `MECH_Resource_System.md` 에서 정의한다.
> 이 문서에서 자원 관련 내용을 직접 기술하지 않는다.

**허기-식량 연동** (NPC 스탯과 직접 연결되는 항목만 명시):
```
식량 1개 소비 → 허기 +1
```
