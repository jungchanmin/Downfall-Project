# 아지트 운영 및 개조 시스템

## 1. 목적

이 문서는 아지트의 고정 공간을 새로 증축하는 대신, 이미 존재하는 방을 목적에 따라 개조하고 탐색에서 획득한 설비를 배치해 생존자 행동과 거점 기능을 변화시키는 공통 규칙을 정의한다.

세부 부품 비용, 행동 수치, 방별 수용 인원, 판정 난이도와 설비 성능은 후속 DB와 테스트 단계에서 조정한다.

## 2. 핵심 원칙

- 아지트의 물리적 방 목록은 `LORE_LOC_Hideout`가 소유한다.
- 시스템 문서는 각 방의 기본 기능, 개조 분기와 행동 슬롯을 소유한다.
- 새 방을 무한 증축하지 않고 기존 방의 용도를 바꾼다.
- 리모델링은 공간과 기본 기능을 만든다.
- 배치 설비는 특별 행동, 보정과 이벤트를 추가한다.
- 한 방을 모든 기능의 최적해로 만들지 않는다.
- 개조 선택은 동료 구성과 플레이 스타일에 따른 빌드 차이를 만들어야 한다.
- R18 모듈은 공통 방을 참조하지만 공통 Hideout System은 R18 문서에 의존하지 않는다.

## 3. 기본 구조

```text
고정 방
→ 기본 행동
→ 개조 방향 선택
→ 시설 단계 향상
→ 설비 배치
→ 자율행동·이벤트·방어 효과 변화
```

각 방은 다음 데이터를 가진다.

```yaml
room_id:
name:
zone:
base_actions: []
capacity:
remodel_options: []
remodel_level:
facility_slots:
installed_facilities: []
condition:
blocked_actions: []
r18_extension: none
```

## 4. 방 상태

| 상태 | 의미 |
|---|---|
| 정상 | 모든 기본·개조 기능 사용 가능 |
| 손상 | 일부 보정 감소, 특정 행동 위험 증가 |
| 봉쇄 | 출입 또는 행동 제한 |
| 파괴 | 핵심 기능 중단, 복구 전 사용 불가 |
| 오염 | 치료·휴식·식량 관련 부작용 가능 |
| 점거 | 괴물·적대 생존자·이상현상이 통제 |

이벤트는 방을 일시적으로 손상·봉쇄·파괴·오염·점거 상태로 만들 수 있다.

## 5. 리모델링과 설비

### 5.1 리모델링

리모델링은 부품과 생존자 행동을 소비해 방의 기본 용도를 바꾸거나 강화한다.

권장 요구 요소:

- 부품
- 제작 또는 기술 판정
- 방 상태 정상화
- 필요 시 선행 개조
- 개조 행동에 참여할 생존자

실패 시 모든 부품을 잃는 구조는 지양한다. 부분 실패는 추가 부품 요구, 행동 소모, 방 손상 또는 지연으로 처리할 수 있다.

### 5.2 설비 배치

설비는 탐색 보상 또는 제작 결과로 획득한다.

- 방의 시설 슬롯에 설치
- 방 유형과 설비 태그가 맞아야 함
- 설치 후 특별 행동 또는 보정 제공
- 이동·철거 가능 여부는 설비별 정의
- 방이 파괴돼도 설비가 반드시 함께 소실되지는 않음

예:

```text
작은방 → 의무실 개조
+ 의료가방 설치
+ 전문 의료장비 설치
→ 일반 치료 / 전문 치료 / 기벽 치료 선택지 해금
```

## 6. 자율행동 공통 규칙

아지트에 있는 생존자는 아침과 저녁에 기본적으로 한 번의 자율행동 또는 명령 행동을 수행할 수 있다.

세부 행동 횟수와 피로 조건은 후속 밸런스 단계에서 확정한다.

### 6.1 기본 행동

| 행동 | 핵심 기능 | 주 스탯 |
|---|---|---|
| 휴식 | 피로·컨디션·정신력 회복 | 없음 또는 방 효과 |
| 훈련 | 경험·숙련·전투 준비 | 훈련 종류별 |
| 대화·교류 | 관계·스트레스·갈등 변화 | 대화 / 매력 |
| 제작·수리 | 장비 제작·수리·설비 설치 | 제작 / 기술 |
| 간호 | 부상·상태이상·기벽 치료 | 간호 |
| 경계 | 불침번 준비·침입 탐지 | 탐색 / 전투 |
| 조사·연구 | 단서 분석·괴물·지역 정보 정리 | 지능 / 탐색 |
| 취미 | 스트레스·관계·고유 능력 해금 | 인물별 |
| 시설 강화·보수 | 방 개조·복구·방어 강화 | 제작 / 기술 |

### 6.2 행동 불능

행동 불능 피로 임계치, 중상, 구금, 특정 기벽 또는 이벤트 상태에서는 선택 가능한 행동이 제한된다.

기본적으로 완전 행동 불능 상태의 생존자는 휴식 또는 치료만 선택할 수 있다.

### 6.3 수용 인원

각 방은 동시에 행동할 수 있는 인원 수를 가진다.

- 기본 수용 인원은 방 크기와 개조에 따라 결정
- 같은 방의 인물끼리 대화·갈등·관계 이벤트 발생 가능
- 수용 한도를 초과하면 행동 효율 감소 또는 선택 불가
- 시설 단계가 수용 인원을 늘릴 수 있음

## 7. 기벽 치료

기벽 치료는 여러 번 채우는 진행도 게이지를 기본으로 사용하지 않는다.

각 기벽은 제거 가능 여부, 난이도, 필요한 의약품·기호품·시설과 특수 조건을 개별 정의한다.

| 치료 등급 | 기본 요구 |
|---|---|
| 쉬움 | 간호 판정 |
| 보통 | 간호 + 일반 의료품 또는 적합한 기호품 |
| 어려움 | 높은 간호 + 전문 의료품 또는 적합한 시설 |
| 전문적 치료 | 전문 의료품 + 높은 간호 + 의무실·특수 설비 |
| 특수 | 이벤트·유물·고유 능력·희귀 설비 |

한 번의 간호 행동에서 성공, 부분 성공 또는 실패를 처리한다.

- 성공: 제거 또는 치료 완료
- 부분 성공: 당일·일정 페이즈 효과 완화, 재시도 보정 또는 자원 일부 보존
- 실패: 행동과 일부 자원 소모

일반 간호 행동이 모든 기벽을 동일하게 제거해서는 안 된다.

## 8. 고정 방과 개조 후보

### 8.1 대문

기본 행동:

- 경계
- 외부 흔적 조사
- 출입 통제
- 응급 보강

개조 후보:

- **강화 관문**: 아지트 침입 지연과 방어 보정
- **검문소**: 귀환자·방문자 검사와 오염 조기 발견
- **함정 진입로**: 침입자 피해·경고, 아군 출입 부담 증가

R18 확장: `optional` — 포획·침입 사건의 추가 대응만 모듈에서 제공

### 8.2 정원

기본 행동:

- 침입 흔적 조사
- 은신·관측
- 간단한 자재 정리

개조 후보:

- **정찰 정원**: 은폐 관측과 외부 접근 탐지
- **훈련 마당**: 전투·투척·기동 훈련
- **완충 지대**: 함정과 장애물로 침입 지연

### 8.3 연못

기본 행동:

- 이상현상 조사
- 투척 훈련

개조 후보:

- **관측 구역**: 수면 이상현상·의식 단서 연구
- **봉쇄 구역**: 위험 접근 차단과 사건 발생 억제

연못을 독립 물 자원의 안정적 생산지로 만들지 않는다.

### 8.4 텃밭

기본 행동:

- 재배
- 토양 조사
- 음식물 처리

개조 후보:

- **생존 텃밭**: 제한적인 식량 보조
- **약초 재배지**: 일반 의료품·상태 치료 재료 후보
- **오염 연구지**: 위험하지만 단서·특수 재료 획득

텃밭은 외출을 대체할 만큼 식량을 생산하지 않는다.

### 8.5 수영장

기본 행동:

- 대형 함정 준비
- 외부 훈련
- 이상현상 조사

개조 후보:

- **방어 함정구역**: 아지트 전투의 지역 통제
- **집단 훈련장**: 다수 생존자 전투 훈련
- **봉쇄 저장구역**: 대형 설비·위험 물품 보관

### 8.6 현관

기본 행동:

- 외출 전 장비 점검
- 귀환 물자 정리
- 오염·부상 확인

개조 후보:

- **출정 준비실**: 장비 배정과 공용 가방 준비 보조
- **귀환 검역실**: 부상·감염·기벽·오염 조기 확인
- **전방 방어실**: 아지트 전투의 첫 방어선

### 8.7 거실

기본 행동:

- 집단 휴식
- 회의
- 대화·교류
- 집단 의사결정

개조 후보:

- **공동생활실**: 집단 휴식·관계 회복·갈등 완화
- **작전실**: 지도·단서·괴물 정보 분석, 외출 준비 강화
- **오락실**: 취미·기호품·스트레스 완화·관계 이벤트
- **집단 훈련실**: 이론·전술·기초 훈련

거실은 대표적인 빌드 분기 방이다. 모든 개조 효과를 동시에 가질 수 없게 한다.

### 8.8 식당

기본 행동:

- 배식
- 단체 식사
- 대화·관계 이벤트

개조 후보:

- **공동 식당**: 배식 질서·집단 스트레스 완화
- **회의 식당**: 세력 협상·방문자 접대·투표 이벤트
- **긴급 배급소**: 식량 손실·도난·폭식 관련 사건 대응

### 8.9 주방

기본 행동:

- 조리
- 식량 정리
- 간단 제작

개조 후보:

- **보존 주방**: 식품 환산 효율·부패 방지
- **대량 조리실**: 집단 배식과 기호식 제공
- **응급 작업대**: 간단 도구·소비품 제작

### 8.10 세탁실

기본 행동:

- 의복 수선
- 붕대·천 세척
- 오염 제거 보조

개조 후보:

- **위생 관리실**: 감염·오염 대응
- **의복 정비실**: 보호 장비·의복 수선
- **재료 재생실**: 천·붕대·필터류 회수

R18 확장: `required` — R18 의복 게이지와 실제 장비 내구도의 복구 연결 인터페이스 필요

### 8.11 작은방

기본 행동:

- 개인 휴식
- 일기·기억 정리
- 은밀한 대화

개조 후보:

- **의무실**: 일반 치료와 의료품 사용
- **상담실**: 정신 기벽·트라우마·관계 안정 치료
- **격리실**: 감염·침식·위험 인물 통제
- **개인 작업실**: 집중 제작·연구·취미

작은방은 전문 역할을 선택하는 핵심 빌드 분기 방이다.

R18 확장: `required` — R18 전용 치료·격리·상담 효과는 확장 모듈이 같은 방 ID를 참조

### 8.12 창고

기본 행동:

- 물자 보관
- 출정 물품 준비
- 함정·부품 정리

개조 후보:

- **정리 창고**: 저장·분류·준비 편의
- **보존 창고**: 식량·의료품 손실 감소
- **위험물 창고**: 폭발물·오염물·유물 격리
- **설비 보관소**: 미설치 아지트 설비 보관

### 8.13 지하실

기본 행동:

- 비상 보관
- 구금
- 최후 대피

개조 후보:

- **비상 대피소**: 아지트 공격 시 생존 보조
- **구금실**: 적대 생존자·위험 인물 통제
- **격리 연구실**: 괴물 부산물·침식·위험 유물 연구
- **비상 창고**: 아지트 파괴 시 일부 자원 보호

R18 확장: `optional` — 전용 포획·침식 사건을 추가할 수 있으나 공통 구금 기능은 완전하게 작동

### 8.14 큰방

기본 행동:

- 숙면
- 개인·은밀한 대화
- 수면 중 기습 대응

개조 후보:

- **공동 침실**: 다수 휴식과 피로 회복
- **회복실**: 중상자 휴식·치료 효율
- **간부실**: 리더·핵심 생존자 지휘·개별 대화
- **고급 훈련실**: 소수 정예 훈련과 고유 능력 해금

큰방도 대표적인 빌드 분기 방이다.

R18 확장: `required` — 사적 관계·수면·전용 회복 이벤트를 확장 가능

### 8.15 베란다

기본 행동:

- 외부 감시
- 신호 교환
- 개인 스트레스 해소

개조 후보:

- **감시초소**: 새벽 맵 괴물·세력 정보 일부 조기 공개
- **통신소**: 외부 세력·무전 이벤트 강화
- **저격·관측 지점**: 아지트 전투와 원거리 관측 보조

### 8.16 샤워실

기본 행동:

- 세척
- 오염 제거
- 상처 세척

개조 후보:

- **제독실**: 강한 오염·감염 대응
- **회복 목욕실**: 피로·스트레스 회복
- **응급 세척실**: 의료 처치 전 준비와 화상·화학 오염 대응

R18 확장: `required` — R18 전용 상태·신체 변이·관계 이벤트는 모듈이 소유

### 8.17 화장실

기본 행동:

- 위생 관리
- 단독 휴식
- 신체 이상 확인

개조 후보:

- **위생실**: 감염·질병 사건 억제
- **검사실**: 신체 이상·기벽 조기 발견
- **봉쇄 구역**: 이상현상 확산 차단

## 9. 시설 단계

권장 상태 표현:

```text
미개조
임시
정비됨
강화됨
```

시설 단계는 다음에 영향을 줄 수 있다.

- 방 수용 인원
- 행동 보정
- 설비 슬롯
- 사건 저항
- 파괴 내성
- 특별 행동 해금

모든 방이 동일한 최대 단계를 가질 필요는 없다.

## 10. 아지트 빌드 원칙

- 모든 방을 한 회차에 완성하기 어렵게 한다.
- 부품·설비·행동 시간의 경쟁을 만든다.
- 동료 약점을 보완하거나 강점을 증폭하는 선택이 가능해야 한다.
- 전투 취약 파티는 감시초소·작전실·함정구역·대피소로 회피·방어 빌드를 만들 수 있다.
- 제작 강점 파티는 작업실·설비 보관소·보존 주방을 강화할 수 있다.
- 관계 중심 파티는 공동생활실·오락실·상담실을 선택할 수 있다.
- 치료 중심 파티는 의무실·회복실·제독실을 연결할 수 있다.

## 11. 파괴·봉쇄·복구

### 11.1 사건 결과

이벤트는 다음 중 하나를 적용할 수 있다.

- 시설 단계 일시 감소
- 설비 사용 불가
- 방 봉쇄
- 방 파괴
- 오염 누적
- 적대 개체 점거

### 11.2 복구

복구는 제작·기술·부품·행동을 요구한다.

```text
상태 확인
→ 위험 제거
→ 부품 확보
→ 제작/기술 판정
→ 방 기능 복구
→ 설비 재설치 또는 수리
```

방이 파괴되더라도 투자 전체가 무조건 소실되지 않도록 한다.

- 일부 설비 회수
- 이전 개조 단계 일부 보존
- 긴급 임시 복구
- 특정 NPC 능력으로 손실 완화

### 11.3 전략적 변화

방 파괴는 단순 벌점이 아니라 플레이 방식 변화를 유도해야 한다.

예:

- 작전실 파괴 → 외출 정보 감소
- 의무실 봉쇄 → 일반 치료만 가능
- 현관 점거 → 귀환 위험 증가
- 공동 침실 파괴 → 휴식 인원 제한

## 12. 아지트 전투 연결

아지트 공격 시 방은 전투 구역이 된다.

- 대문·현관: 첫 방어선
- 정원·수영장: 함정·지연
- 거실·복도: 중앙 교전
- 지하실: 대피·최후 방어
- 베란다: 관측·원거리 지원

방 개조는 전투 선택지와 전장 효과를 제공하지만 전투 자체의 판정 규칙은 Combat System이 소유한다.

## 13. 이벤트 연결

방은 다음 이벤트 조건을 제공한다.

```yaml
required_room:
required_remodel:
required_facility:
room_condition:
occupants:
time_phase:
r18_required:
```

이벤트는 방의 이상현상 후보를 사용해 개조 기능과 위험을 동시에 흔들 수 있다.

## 14. R18 확장 규칙

- 공통 방·개조·설비 ID를 그대로 참조한다.
- R18 전용 효과는 모듈 문서가 소유한다.
- R18 OFF에서는 공통 행동과 개조가 정상 작동한다.
- R18 전용 개조를 공통 필수 테크로 만들지 않는다.
- R18 기벽 치료는 개별 제거 가능 여부와 난이도를 따른다.
- 의복 게이지는 실제 장비 내구도와 분리한다.

후속 R18 Hideout 확장 후보:

- 사적 관계 행동
- R18 기벽 치료·완화
- R18 전용 상담·격리
- 의복·신체 상태 정비
- 전용 방 이벤트
