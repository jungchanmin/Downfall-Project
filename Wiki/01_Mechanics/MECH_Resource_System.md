# 📦 자원 체계 시스템

---

## SECTION 1 — 자원 체계 개요

자원은 9종으로 구분되며 크게 3가지 성격으로 분류된다.

| 분류 | 자원 종류 | 성격 |
|---|---|---|
| **생존 자원** | 식량 / 부품 / 장비 / 유물 | 직접 소비·활용하는 물질적 자원 |
| **사회 자원** | 금화 / 명성 / 단서 / 세력평판 | 사회적 관계·거래에서 작동하는 추상적 자원 |
| **인적 자원** | 동료(생존자) | NPC 무리의 규모·전력을 결정하는 인적 자원 |

---

## SECTION 2 — 생존 자원

### 2-1. 식량 (Food)

```
관리 방식: 정수값
획득 단위: 1개
소비 단위: 1개 / 끼니
```

**역할**: 생존자의 허기 해소. 생존 필수품.

**핵심 규칙**:
- 한 끼에 식량 1개 소비
- 한 번에 대량 섭취 불가 (허기 회복 상한: 끼니당 +1)
- 허기가 0이 되면 체력 감소 시작

**연동 스탯**: 허기 (MECH_NPC_Stats_System SECTION 2-4)

---

### 2-2. 부품 (Parts)

```
관리 방식: 정수값
```

**역할**: 장비 제작·수리·시설 강화의 재료.
완제품(장비)을 만들기 위한 원자재.

**사용처**:
- 제작 스탯 판정 + 부품 소비 → 장비 생산
- 시설 강화·아지트 수리
- 함정·방어 설비 구축

**연동 스탯**: 제작 (MECH_NPC_Stats_System SECTION 3-5)

---

### 2-3. 장비 (Equipment)

```
관리 방식: 종류별 개별 아이템
사용 횟수: 아이템별 상이 (소모성)
```

**역할**: 완제품 아이템. 전투·생존·치료·탐색 등 다양한 상황에서 직접 사용.

**세부 분류**:

| 분류 | 설명 | 예시 |
|---|---|---|
| **의약품** | 체력 회복, 상태이상 치료 | 붕대, 항생제, 진통제 |
| **무기류** | 전투력 보정, 공격 수단 | 칼, 총기, 폭발물 |
| **일회용품** | 단발성 효과 아이템 | 연막탄, 미끼, 응급키트 |
| **사치품** | 스트레스·컨디션 회복 | 술, 담배, 사탕 |

**구역별 획득 경향**:
```
의약품: 병원·약국·아지트 창고 구역
무기류: 경찰서·군사 구역·암시장
일회용품: 일반 상점·주거 구역
사치품: 상점가·식당·특수 구역
```

> ⚠️ 세부 아이템 데이터베이스는 별도 문서 `ITEM_Equipment_DB.md` 에서 관리 예정.

---

### 2-4. 유물 (Relic)

```
관리 방식: 종류별 개별 아이템
사용 횟수: 무제한 (귀속)
획득 난이도: 매우 어려움
```

**역할**: 영구 귀속 아이템. 강력하지만 획득이 극도로 어렵고
대부분 하이리스크·하이리턴 구조.

**장비와의 차이점**:

| 항목 | 장비 | 유물 |
|---|---|---|
| 사용 횟수 | 제한 있음 | 무제한 |
| 귀속 여부 | 거래 가능 | 귀속 (거래 불가) |
| 획득 방법 | 탐색·제작·구매 | 특수 이벤트·보스 처치·숨겨진 장소 |
| 리스크 | 낮음~중간 | 높음~극한 |

> ⚠️ 유물 데이터베이스는 별도 문서 `ITEM_Relic_DB.md` 에서 관리 예정.

## SECTION 3 — 사회 자원

### 3-1. 금화 (Gold)

```
관리 방식: 정수값
주요 사용처: 암시장 거래
```

**역할**: 암시장·특수 거래에서 통용되는 화폐.
일반 탐색보다 암시장에서 희귀 아이템을 안정적으로 획득하는 수단.

**획득 경로**:
- 특수 구역 탐색
- 특정 이벤트 보상
- 세력 NPC 거래

**사용처**:
- 암시장 장비·유물 구매
- 세력 NPC 협상·뇌물
- 특수 서비스 구매

---

### 3-2. 명성 (Reputation)

```
관리 방식: 정수값 (내부 계산)
출력 방식: 등급 텍스트로 변환 출력
```

**역할**: 생존자 무리의 사회적 위상. 엔딩 조건과 직결.

**획득 조건**:
- 어려운 구역 제패 보상
- 멸망의 괴물 처치
- 특별한 업적 달성
- 세력 NPC 대형 의뢰 완수

**등급 출력 기준 (임시)**:

| 내부 수치 | 출력 텍스트 |
|---|---|
| 0 ~ 9 | 무명: 당신들의 이름은 아직 알려지지 않았습니다. |
| 10 ~ 29 | 생존자: 이 마을에서 살아남은 이들이 있다는 소문이 돕니다. |
| 30 ~ 59 | 용사: 당신들의 활약이 생존자들 사이에서 회자됩니다. |
| 60 ~ 89 | 영웅: 당신들의 이름은 이 지옥에서 희망의 상징이 되었습니다. |
| 90 ~ 119 | 전설: 당신들의 이야기는 살아남은 모든 이들의 입에서 오르내립니다. |
| 120+ | 초월: 어딜 가나 당신과 당신의 동료를 알아봅니다. |

> ⚠️ 등급 수치 기준은 게임 밸런스 확정 후 조정 예정.

---

### 3-3. 단서 (Clue)

```
관리 방식: 종류별 개별 관리
```

**역할**: 다양한 목적에 활용되는 정보 자원.

**사용처**:
- 괴물 약점 파악·공략
- 유물 위치·획득 조건 해제
- 세력 NPC 접근 조건 충족
- 세력 우호도 향상
- 숨겨진 이벤트·루트 해제

**획득 경로**:
- 탐색 이벤트 성공
- 특정 NPC와의 대화
- 도감·기록물 해독
- 괴물 관찰·분석

> ⚠️ 단서 종류별 데이터베이스는 별도 문서에서 관리 예정.

---

### 3-4. 세력 평판/호감도 (Faction Reputation)

```
관리 방식: 세력별 개별 정수값
범위: -100 ~ +100
기준값: 0 (중립)
```

**역할**: 각 세력과의 관계 지표.
세력 관련 이벤트·구역에서 판정 modifier 및 분기 조건으로 작용.

**임계값**:

| 수치 | 관계 상태 |
|---|---|
| -100 ~ -60 | 적대: 해당 세력 구역 진입 시 즉시 전투 |
| -59 ~ -20 | 경계: 거래·협력 불가. 이벤트 불이익 |
| -19 ~ +19 | 중립: 기본 거래 가능 |
| +20 ~ +59 | 우호: 특수 거래·의뢰 접근 가능 |
| +60 ~ +100 | 동맹: 세력 지원·합동 작전 가능 |

**변동 조건**:
- 세력 의뢰 완수: +N
- 세력 NPC 사망·배신: -N
- 세력 구역 약탈: -N
- 단서 제공·협력: +N

## SECTION 4 — 인적 자원

### 4-1. 동료 (Survivor)

```
관리 방식: NPC 개별 관리
```

**역할**: 생존자 무리에 새로운 동료를 편입시키는 자원.
무리의 전력·역할 분담을 결정하는 핵심 자원.

**획득 경로**:
- 구역 최초 탐험 성공 보상
- 특수 구조 이벤트
- 세력 NPC 합류 조건 충족
- 특정 스토리 이벤트

**편입 조건**:
- 아지트 수용 인원 초과 여부 확인
- 신규 NPC 신뢰도·적응도 초기값 설정
- 기존 생존자와의 관계 초기화

**편입 후 관리**:
- 신규 NPC 로어 바이블 등재 필요
- 기본 스탯·능력 스탯 초기값 확정
- 성격 스탯 고정값 확정

## SECTION 5 — 자원 관리 원칙

### 5-1. 이벤트 연동 원칙

```
자원 획득·소비는 각 이벤트 파일의 effects 블록에서 개별 관리.
이 문서는 자원의 정의·범위·역할만 규정한다.
```

### 5-2. 1회 변동 상한

```
식량:     ±3
부품:     ±3
금화:     ±5 (암시장 거래 특성상 상한 완화)
명성:     +1 ~ +10 (업적 규모에 따라 차등)
세력평판: ±10 (1회 이벤트 기준)
```

> 장비·유물·단서·동료는 개수 단위 관리.
> 1회 획득 상한은 이벤트 설계 시 디렉터 판단.

### 5-3. 자원 부족 임계값

```
식량 = 0:     다음 페이즈부터 전 생존자 허기 감소 없음
              → 체력 감소 이벤트 강제 발동
부품 = 0:     제작·수리 불가
금화 = 0:     암시장 거래 불가
동료 = 1명:   리더 단독 상태. 특수 이벤트 발동 가능
```

## SECTION 6 — 기벽 (Quirk)

> ⚠️ 기벽은 자원이 아닌 **NPC 상태이상의 특수 분류**다.
> 자원 체계와 혼용하지 않는다.

```
획득 조건:
  - 같은 종류의 F5_Cosmic 이벤트 2회 이상 경험
  - 정신력 임계값 이하 하락
  - 특수 트라우마 이벤트 fail 결과

관리 위치: LORE_CHAR_{NPC}.md 의 emits 및 상태이상 항목
```

**기벽과 자원의 차이**:

| 항목 | 자원 | 기벽 |
|---|---|---|
| 성격 | 획득·소비·거래 가능 | NPC에 귀속. 제거 어려움 |
| 관리 | 수치로 관리 | 서술적 상태로 관리 |
| 영향 | 외부 상황에 작용 | NPC 내면·행동에 작용 |
