# ⚖️ DOWNFALL 이벤트 보상 책정기 (v1.0)

> **"이벤트는 이미 쓰여졌다. 나는 그 결과에 값을 매긴다."**

이 문서는 **완성된 이벤트 출력물**(Gemini가 생성한 텍스트와 선택지)을 입력으로 받아,
그 이벤트에 부여될 스탯 변동·기벽·자원·판정 난이도를 **역방향으로 산정**하는 절차를 규정한다.
파이프라인의 *입력*이 아니라 *출력물에 대한 검수·책정*이라는 점이 핵심이다.

---

## SECTION 0 — 대원칙 (Override Rules)

> **이 섹션은 하위 모든 섹션을 override 한다.**

### 0-1. 성공·실패 보상 대원칙

| 분기 | 규칙 |
|---|---|
| **성공** | 반드시 **순보상(net positive)**. 보상 0 금지. |
| **실패** | 반드시 **순처벌(net negative)**. 보상 부여 금지. |

- 성공 분기에 미약한 패널티(T1)를 붙이는 것은 허용된다. **단, 보상이 그것을 명백히 상회할 때만.**
- 실패 분기에 보상을 섞는 것은 **금지**한다. (실패에 위안을 주지 않는다.)
- "성공해도 실패해도 패널티"는 **명백한 위반**이다.

### 0-2. 내러티브 구속 원칙

- 텍스트가 **명시적으로 묘사한 결과**(예: "발목을 접질렸다")는 수치로 **반드시 반영**한다.
- 책정기는 텍스트에 **없는 것을 창작하지 않는다.** 텍스트가 암시한 것을 수치화할 뿐이다.
- 단, 검수 단계에서 구체적 신체 손상을 텍스트가 *확정*하는 서술은 **회피**를 권장한다.
  (구체 기벽은 보상으로 부여하되, 계열·개수까지만 산정 — SECTION 6 참조)

### 0-3. 역할 경계

- 본 문서는 **수치·구조·난이도 산정**만 다룬다.
- 텍스트의 문체·어휘·수위 평가는 **하지 않는다.** (`SYS_Downfall_AI_Assistant` SECTION 0 준수)

## SECTION 1 — 책정 5단계 절차 + 2경로 라우팅

### 1-1. 책정 5단계

```
1. 맥락 태깅       : 텍스트에서 actor / phase / location / track / family 추출
2. 강도 밴드 산정  : 위험 점수 합산 → B1~B4 (SECTION 2)
3. 경로 라우팅     : 무판정 / 판정 경로 결정 (SECTION 1-2)
4. 채널·판정 설계  : 사다리 등급 매핑 + 복합 스탯 판정 (SECTION 3·5)
5. 수치 확정       : 허용 등급 내에서 4단 분기 작성 (SECTION 4)
```

### 1-2. 2경로 라우팅

밴드 산정 후, 이벤트는 둘 중 하나의 경로로 분기된다.

| 경로 | 조건 | 처리 |
|---|---|---|
| **무판정 (No-Check)** | 밴드 B1~B2 · 선택지 없음 · 돌발/일상 | 출력물 + **즉시 단일 효과**. 한 방향(T1~T2 또는 R1~R2)만. **성패 개념 없음.** |
| **판정 (Check)** | 밴드 B3~B4 · 선택지 존재 · 탐색/탐험/데이 | 선택지 + 복합 스탯 판정 + **성패 4단 분기** (대원칙 적용) |

> 무판정 경로는 "분위기 효과"다. 가벼운 돌발 이벤트에 판정·선택지를 강제하지 않는다.
> 판정 경로는 난이도가 실제로 의미를 갖는 고비용 이벤트 전용이다.

## SECTION 2 — 강도 밴드 산정 (가산점 모델)

재량을 배제하고 규칙으로 고정하기 위해 **결정론적 가산점**을 사용한다.

### 2-1. 위험 점수 산정

| 축 | 점수 |
|---|---|
| **페이즈** | 아침·점심 0 · 새벽·저녁 1 · 밤 2 |
| **장소** | 아지트·성당 0 · 상점가 1 · 폐가·숲·비밀연구소 2 |
| **패밀리** | F1 0 · F2·F3 1 · F4 2 · F5 3 |
| **트랙** | A 0 · B +1 |

### 2-2. 밴드 결정

```
합계 0~1 → B1 경미
합계 2~3 → B2 보통
합계 4~5 → B3 심각
합계 6+  → B4 파국
```

### 2-3. 성격 스탯 미세 조정

> NPC 성격 스탯은 **밴드를 변경하지 못한다.** 스코프 폭발 방지.

- 신경성 높음(7~10): 스트레스 채널 수치만 +1
- 인내심 높음(7~10): 스트레스 채널 수치만 −1
- (밴드 등급·기벽 부여 여부에는 영향 없음)

## SECTION 3 — 패널티/보상 이중 사다리

**High-Risk → High-Return** 원칙을 구조로 강제한다.
실패는 패널티 사다리에서, 성공은 보상 사다리에서만 채널을 고른다.

### 3-1. 패널티 사다리 (실패 분기 전용)

| 등급 | 채널 |
|---|---|
| **T1 미약** | 허기 +1 · 컨디션 −1 · 피로도 +1 · 욕구불만 +1 · 상태이상(서술) · 친밀도 −1 |
| **T2 경** | 스트레스 +1~2 |
| **T3 중** | 스트레스 +3 · 자원 소량 −1~2 · 능력 일시 디버프 |
| **T4 강** | 정신력 −1~2 · 불만도 +1~2 · 체력 − · 자원 다량 −3 |
| **T5 파국** | 기벽 +1 · 괴물 출현 · 부상 상태(체력≤20) 진입 |

> 강도 서열: 정신력 > 스트레스 · 체력 > 허기 · 불만도 ≫ 상태이상·욕구불만.
> 불만도가 T4인 이유: 리더 절대 권한·명령 거부·이탈 체계에 직결되기 때문.

### 3-2. 보상 사다리 (성공 분기 전용)

| 등급 | 채널 |
|---|---|
| **R1 미약** | 스트레스 −1 · 컨디션 +1 · 친밀도 +1 · 피로도 −1 |
| **R2 경** | 식량·부품 +1~2 · 허기 회복 |
| **R3 중** | 자원 다량 +3 · 금화 · 단서 |
| **R4 강** | 장비 · 세력평판 + |
| **R5 파국급** | 유물 · 명성 · 동료 합류 |

### 3-3. 밴드 → 사다리 매핑

| 밴드 | 무판정 즉시효과 | 성공(Check) | 실패(Check) |
|---|---|---|---|
| **B1** | R1 또는 T1 (단일·경미) | R1~R2 | T1 |
| **B2** | R2 또는 T2 | R2~R3 | T2 |
| **B3** | (대개 판정 경로) | R3~R4 | T3~T4 |
| **B4** | (판정 경로) | R4~R5 | T4~T5 |

> 자원 변동 상한은 `MECH_Resource_System` SECTION 5-2를 따른다 (식량·부품 ±3, 금화 ±5, 세력평판 ±10).
> 스탯 변동 상한은 `MECH_NPC_Stats_System` SECTION 6-4를 따른다 (기본 스탯 ±3, 능력 ±1, 친밀도 ±1, 침식도 ±3).

## SECTION 4 — 성패 4단 구조

판정 경로(Check)의 결과는 4단으로 분기한다.

### 4-1. 4단 정의

| 분기 | 판정 결과 | 산출 |
|---|---|---|
| **대성공** | DC를 큰 폭으로 초과 달성 | 보상 사다리 **상단** · 패널티 0 |
| **일반성공** | DC 달성 | 보상 사다리 **하단** · (옵션) 미약 패널티 T1 |
| **실패** | DC 미달 | 패널티 사다리 **하단** · 보상 0 |
| **대실패** | DC를 큰 폭으로 미달 | 패널티 사다리 **상단** · B3+ 시 기벽/괴물 발동 가능 |

> 주사위 체계가 미확정이므로 "큰 폭"의 절대 기준은 추후 확정한다.
> 잠정: 대성공 = DC+5 이상 초과, 대실패 = DC−5 이상 미달.

### 4-2. 셜리 잭슨 정합

- 대성공이라도 **코어 어긋남 현상 자체는 해소되지 않는다.** 보상은 "단서·이해"에 그친다.
- 일반성공은 "서늘한 확인", 실패는 "놓친 무언가"의 톤을 유지한다. (텍스트는 Gemini 소관, 책정기는 수치만)

## SECTION 5 — 복합 스탯 판정 + DC 배정

### 5-1. 맥락 기반 스탯 선정

선택지 라벨의 표면 키워드가 아니라 **이벤트 맥락**으로 1~2개 스탯을 선정한다. 중복 허용.

```
"대화로 해결한다"        → 주: 대화   + 보조: 매력
"무력으로 돌파한다"      → 주: 전투   + 보조: 탐색(기술)
"조용히 살핀다"          → 주: 탐색   + 보조: 지능
"치료한다"               → 주: 간호   + 보조: 지능
"교섭한다" (R18)         → 주: 매력   + 보조: 침식도
```

### 5-2. 복합 판정식

```
주 스탯으로 DC 판정. 보조 스탯이 modifier 제공.
  보조 스탯 ≥ 5 → modifier +1
  보조 스탯 ≥ 8 → modifier +2
```

> 정밀 환산은 주사위 시스템 도입 후 재조정 (`MECH_NPC_Stats_System` SECTION 6 연동).

### 5-3. 밴드별 DC

```
B3 → DC 12
B4 → DC 13 ~ 15  (F5_Cosmic 기본값 13)
```

> `MECH_NPC_Stats_System` SECTION 6-2 DC 기준표 일치.
> modifier 누적 상한 ±5 준수.

## SECTION 6 — 기벽 부여 규칙

### 6-1. 부여 조건

```
기벽 부여 = (판정 경로 대실패 분기) AND (밴드 B3 이상)
```

- 무판정 경로·성공 분기에서는 기벽을 부여하지 않는다.
- 책정기는 **계열과 개수만** 산정한다: `기벽 +1 (육체)` / `기벽 +1 (정신)`.
- 구체 항목(어느 DB 항목인지)은 이벤트 맥락에 따라 후속 결정한다. (책정기는 항목 확정 안 함)

### 6-2. 계열 결정 트리

```
직접 충돌(생존자·괴물과의 물리적 접촉)  ──→  육체 계열 (MECH_Quirk_Physical_DB)
이상현상(코즈믹 누설·공간의 악의)        ──→  정신 계열: 정신이상형 (MECH_Quirk_Mental_DB)
생존자간 감정 마찰·관계 파탄            ──→  정신 계열: 트라우마/욕구폭발형
R18 트랙 + 침식 진행                    ──→  R18 계열 (MECH_Quirk_R18_DB)
```

### 6-3. 중복·상충 금지

- 동일 NPC가 이미 보유한 기벽은 재부여하지 않는다. (trigger의 `not` 조건으로 차단)
- 상호 배타 기벽(예: 거식증 ↔ 폭식증) 동시 부여 금지.
- 부여 시 `Flag_Memory_{actor}_{event_id}`를 함께 emit하여 출처를 추적 가능하게 한다.

## SECTION 7 — 자원 보상 분기

성공 분기에서 자원을 줄 때, 이벤트 맥락으로 자원 종류를 결정한다.

| 이벤트 맥락 | 우선 자원 |
|---|---|
| 식료품·주방·농장 탐색 | 식량 |
| 공방·기계·잔해 탐색 | 부품 |
| 암시장·시신·금고 | 금화 |
| 무기고·의무실 | 장비(무기류/의약품) |
| 보스 처치·숨겨진 장소·특수 이벤트 | 유물 |
| 정보·기록물·관찰·해독 | 단서 |
| 세력 의뢰·교류 | 세력평판 |

- 1회 변동 상한은 `MECH_Resource_System` SECTION 5-2를 따른다.
- 장비·유물·단서·동료는 개수 단위 — 1회 1개를 기본으로 하고 초과 시 디렉터 판단.

---

## SECTION 8 — 괴물·세력 훅 (미규격화)

> ⚠️ 괴물·세력 데이터는 게임 디자인상 **존재하지만**, 위키에 **규격화 문서가 아직 없다.**
> 따라서 본 문서는 **연결 지점(hook)만** 정의하고, 실제 수치는 후속 문서 작성 후 연결한다.

| 훅 | 발동 조건 | 미작성 항목 |
|---|---|---|
| **괴물 출현** | T5 (B4 대실패) | `MONSTER_DB.md` — 개체별 스탯·약점·드롭 |
| **세력평판 변동** | F4_Encounter 맥락 | `FACTION_DB.md` — 세력 목록·우호도 변동표 |

- 책정기는 "이 분기에서 괴물이 출현한다 / 세력평판이 변동한다"까지만 명시한다.
- 구체 개체·세력·수치를 **상상으로 끼워 넣지 않는다.** (정합성 파괴 방지)
- `MONSTER_DB` / `FACTION_DB` 작성 완료 시 본 SECTION을 수치 연동으로 갱신한다.

## SECTION 9 — 워크드 예시

### 9-1. B1 무판정 — 아침/아지트/돌발

입력 발췌: *아침. 아지트 거실. 레이첼이 창틀에 앉아 식은 차를 마신다. 별일 없는 정적.*

```
맥락: 아침0 + 아지트0 + F2 1 + TrackA 0 = 1점 → B1
경로: 무판정 (선택지 없음, 돌발)
산출: 즉시 단일 효과 — R1
effects:
  "Rachel.스트레스": -1   # 잠깐의 휴식. 한 방향, 성패 없음.
```

### 9-2. B3 판정 — 저녁/폐가/탐색

입력 발췌: *저녁. 폐가 1층. 무너진 찬장 뒤에서 봉인된 보급 상자가 보인다.
(A: 힘으로 뜯어낸다 / B: 조심스레 잠금을 살핀다)*

```
맥락: 저녁1 + 폐가2 + F1 0 + TrackB 1 = 4점 → B3
경로: 판정
choices:
  - id: ch_force      # 주:전투 보조:제작, DC12
    대성공: { resource: { 부품: +3 }, flags_emit: [Equip_보급상자] }   # R4
```
