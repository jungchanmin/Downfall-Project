# 아이템 획득 및 지역 보상 분배 시스템

## 1. 목적

이 문서는 탐험 중 아이템과 자원이 어떤 구조로 등장하는지 정의한다.

개별 아이템마다 장소별 고정 확률을 부여하지 않는다. 획득 구조는 다음 네 요소의 결합으로 처리한다.

```text
카테고리 기본 등장도
+ 지역 특성 보너스
+ 지역 위험도 보정
+ 잔존 자원도 보정
```

실제 퍼센트, 세부 수량과 아이템 등급 수치는 게임 테스트와 밸런스 패치 단계에서 확정한다.

## 2. 핵심 원칙

- 개별 아이템이 아니라 카테고리를 먼저 추첨한다.
- 장소는 개별 아이템 확률이 아니라 카테고리 적합도를 가진다.
- 위험한 지역일수록 보상 슬롯과 고급 카테고리 접근성이 증가한다.
- 안전한 지역도 고갈 후 최소한의 기본 자원은 획득할 수 있다.
- 잔존 자원도는 획득 가능 여부보다 추가 보상과 희귀 보상의 효율을 조정한다.
- 중요 물품과 유물은 일반 랜덤 테이블이 아니라 이벤트·보상 데이터가 소유한다.

## 3. 처리 순서

```text
1. 탐색 결과가 보상 발생 여부를 결정
2. 보상 슬롯 수 결정
3. 각 슬롯의 아이템 카테고리 결정
4. 지역 특성 보너스 적용
5. 위험도에 따라 보상 품질 또는 고급 카테고리 전환
6. 잔존 자원도에 따라 추가 슬롯·추가 수량 보정
7. 해당 카테고리의 실제 후보 중 하나를 선택
8. 이벤트·단서·장비 효과에 따른 추가 보상 적용
```

## 4. 카테고리 기본 등장도

| 카테고리 | 기본 등장도 | 설계 의도 |
|---|---|---|
| 식품 | 높음 | 생존 루프를 유지하는 기본 보상 |
| 부품 | 높음 | 여러 장소의 생활·산업 잔해에서 획득 가능 |
| 일반 의료품 | 중간 | 기본 치료에 필요하지만 식량보다 희소 |
| 전문·응급 의료품 | 낮음 | 고성능이며 장소 특성 의존도가 높음 |
| 근접 무기 | 중간 | 생활 도구와 임시 무기 포함 |
| 총기 | 낮음 | 강력한 전투 성능과 경제적 가치 |
| 탄약 | 낮음 | 총기 사용을 제한하는 공용 자원 |
| 보호 장비 | 낮음 | 장기 생존력과 특정 위험 대응을 강화 |
| 유틸리티 장비 | 중간 | 탐색 선택지와 판정 보정을 확장 |
| 전술품 | 낮음 | 전투·탈출·교란에 강한 일회성 보상 |
| 기호품 | 중간 | 관계·스트레스·거래에 사용 |
| 환금품 | 낮음 | 금화 경제와 특수 거래에 사용 |
| 아지트 설비 | 낮음 | 아지트 빌드와 특수 행동을 확장 |
| 중요 물품 | 이벤트 전용 | 일반 랜덤 드롭 금지 |
| 유물 | 희귀·이벤트 전용 | 특수 사건·보스·숨은 구역 전용 |

`높음 / 중간 / 낮음 / 희귀`는 개별 아이템의 절대 확률이 아니라 카테고리 슬롯 선택 경향을 뜻한다.

## 5. 지역 특성 보너스

장소는 다음 형식의 `location_loot_profile`을 가진다.

```yaml
location_id:
primary_categories: []
secondary_categories: []
minor_categories: []
restricted_categories: []
special_reward_sources: []
```

### 5.1 보너스 단계

| 단계 | 의미 |
|---|---|
| 주요 | 해당 장소의 대표 보상. 등장도 상승 |
| 보조 | 합리적으로 획득 가능한 보상. 소폭 상승 |
| 소수 | 등장 가능하지만 장소 대표성은 낮음 |
| 제한 | 일반 탐색에서는 거의 등장하지 않음 |
| 특수 | 특정 이벤트·구역·단서로만 획득 |

### 5.2 장소별 초기 프로필

| 장소 | 주요 | 보조 | 제한 또는 특수 |
|---|---|---|---|
| 병원 | 일반·전문 의료품 | 식품, 기록물, 의료 유틸리티 | 총기, 중장비 |
| 경찰서 | 무기, 탄약, 보호 장비 | 의료품, 통신·관측 장비 | 일반 식품 |
| 철물점 | 부품, 유틸리티, 근접 무기 | 보호 장비, 전술품 | 전문 의료품 |
| 백화점 | 식품, 기호품, 생활품 | 보호 장비, 유틸리티 | 총기 |
| 주거단지 | 식품, 기호품 | 일반 의료품, 임시 무기, 생활 설비 | 전문 장비 |
| 공장 | 부품, 보호 장비 | 유틸리티, 근접 무기, 아지트 설비 | 식품, 의료품 |
| 선착장 | 식품, 로프·공구류 | 근접 무기, 통신·신호 장비 | 전문 의료품 |
| 연구소 | 전문 의료품, 특수 장비, 단서 | 상태이상 치료품, 유틸리티 | 일반 식품 |
| 유흥가 | 기호품, 환금품 | 의료품, 전술품 | 부품 |
| 성당 | 식품, 기록물, 관계 물품 | 의료품 | 총기, 산업 부품 |

장소별 세부 프로필은 각 Location 문서가 선언하고, 이 문서는 공통 해석 규칙을 소유한다.

## 6. 지역 위험도 보정

위험도는 모든 카테고리 수량을 동일하게 올리지 않는다.

위험도는 다음 두 요소를 중심으로 보정한다.

- 보상 슬롯 수
- 고급 카테고리 또는 희귀 후보로 전환될 가능성

| 위험도 | 보상 슬롯 경향 | 품질·희귀도 경향 | 설계 의도 |
|---|---|---|---|
| 안전 | 적음 | 일반 중심 | 기본 생존 자원 확보 가능 |
| 보통 | 보통 | 일반·개선품 혼합 | 표준 탐색 위험과 보상 |
| 위험 | 많음 | 전문·고성능 후보 증가 | 성장 장비와 설비 확보 |
| 극위험 | 많음 또는 특수 | 희귀·전문 중심 | 핵심 성장과 고유 보상 |

### 6.1 위험도 보정 금지 사항

- 위험 지역이라는 이유만으로 모든 기본 식량을 과다 지급하지 않는다.
- 극위험 지역이 단순 물량 파밍 장소가 되지 않도록 한다.
- 위험도는 장소 특성과 무관한 카테고리를 무조건 생성하지 않는다.
- 희귀 장비는 위험도만으로 확정되지 않으며 적합한 장소·구역·이벤트가 필요하다.

## 7. 잔존 자원도

### 7.1 상태 단계

| 상태 | 효과 |
|---|---|
| 풍부 | 기본 보상 외 추가 슬롯 또는 추가 수량 가능성 증가 |
| 보통 | 기본 보상 구조 유지 |
| 감소 | 기본 보상은 유지하지만 추가 보상 효율 감소 |
| 고갈 | 최소 기본 보상은 유지. 추가·고급 보상 효율이 크게 감소 |

### 7.2 고갈 원칙

`고갈`은 자원을 전혀 얻을 수 없다는 뜻이 아니다.

고갈 지역에서도 다음은 가능하다.

- 소량의 식량·부품 등 기본 자원
- 지역 필수 이벤트 보상
- 숨은 구역 보상
- 고유 단서가 지정한 보상
- 맵 배치 괴물 처치 보상
- 세력·세계 이벤트로 새로 유입된 물자

고갈 상태에서 주로 감소하는 것은 다음이다.

- 추가 보상 슬롯
- 추가 수량
- 고급 카테고리 전환
- 일반 탐색으로 얻는 희귀 장비

### 7.3 잔존 자원도 변화

잔존 자원도는 다음 요소로 변할 수 있다.

- 플레이어의 반복 탐색
- 다른 생존자·세력의 약탈
- 지역 파괴
- 세계 이벤트
- 보급품 유입
- 숨은 구역 개방

구체 감소량과 회복 규칙은 후속 밸런스 단계에서 확정한다.

## 8. 보상 슬롯

보상 슬롯은 개별 아이템 하나 또는 하나의 묶음을 생성하는 단위다.

예:

```text
식품 슬롯
→ 통조림 묶음 또는 비스킷 등

부품 슬롯
→ 부품 수량

의료품 슬롯
→ 일반 의료품 후보 1종

장비 슬롯
→ 해당 카테고리 장비 후보 1종
```

### 8.1 슬롯 생성 요인

- 탐색 행동 결과
- 지역 위험도
- 잔존 자원도
- 세부 구역
- 선택한 탐색 방식
- 생존자 탐색 스탯
- 장비 태그
- 단서
- 지역 이벤트 결과

### 8.2 중복 보상 제어

하나의 탐색 결과에서 같은 카테고리가 여러 번 선택될 수 있으나, 다음 중 하나로 처리한다.

- 수량 증가
- 다른 하위 아이템 후보 선택
- 상위 품질로 전환
- 추가 슬롯 유지

구체 방식은 카테고리별 DB가 정의한다.

## 9. 탐색 방식과 보상

### 빠르게 탐색

- 적은 시간 소모
- 기본 보상 중심
- 숨은 보상 발견률 낮음
- 위험 조기 발견 능력 감소 가능

### 신중하게 탐색

- 시간·피로 소모 증가
- 추가 슬롯 또는 단서 발견 가능
- 환경 위험과 매복 탐지에 유리

### 특정 카테고리 집중 탐색

- 선택한 카테고리 등장도 상승
- 다른 카테고리 기회 감소
- 장소의 `primary / secondary` 범위 안에서만 효과적

### 경계하며 탐색

- 보상 효율 감소 가능
- 괴물·매복·환경 위험 조기 대응
- 도주·방어 선택지 강화

## 10. 생존자 스탯과 장비 연결

탐색 보상은 `탐색` 스탯을 중심으로 판정한다.

- 탐색: 보상 발견과 숨은 구역 탐지
- 지능: 발견한 물건의 가치·용도 판별, 추가 단서
- 기술: 기계·전자·잠금 보상 접근
- 제작: 해체·재활용·부품 회수 품질
- 간호: 의료품과 환자 기록의 판별

장비 태그는 보상 슬롯을 직접 무한 생성하지 않고, 특정 보상 접근과 결과 품질에 영향을 준다.

예:

```text
PRY / TOOLKIT
→ 잠긴 보관함 접근

OBSERVE
→ 위험 노출 전에 보상 위치 확인

HACK
→ 전자 잠금 보상 접근

TREAT
→ 의료품 오염·손상 여부 판별
```

## 11. 필수 이벤트와 괴물 보상

지역 필수 이벤트와 맵 배치 괴물의 보상은 일반 탐색 드롭과 분리할 수 있다.

### 필수 이벤트 보상

- 지역 초회 보상
- 고유 단서
- 숨은 구역 해금
- 카테고리 보상 슬롯
- 세력·명성 변화

### 맵 배치 괴물 보상

- 괴물 고유 부산물
- 지역 위험 완화
- 잠긴 탐색 구역 개방
- 고급 카테고리 보상 접근

맵 배치 괴물 처치가 항상 아이템을 직접 떨어뜨릴 필요는 없다. 지역을 안전하게 만들거나 봉쇄된 보상 구역을 개방하는 방식도 허용한다.

## 12. 아지트 설비 연결

아지트 설비는 일반 장비와 별도의 보상 카테고리로 취급한다.

- 발견 시 휴대 가능한 설비 아이템 형태
- 귀환 후 특정 방에 설치
- 설치 전에는 공용 가방 또는 운반 부담 적용 가능
- 설치 후 자율행동·시설 효과·이벤트를 해금

설비의 실제 설치·철거·파괴·복구 규칙은 `MECH_Hideout_System.md`가 소유한다.

## 13. R18 모듈 연결

R18 전용 아이템은 공통 Loot 테이블에 직접 등록하지 않는다.

R18 ON에서는 다음 방식으로 확장한다.

- 특정 장소의 R18 전용 카테고리 보너스 추가
- 공통 카테고리 슬롯을 R18 후보로 일부 대체
- R18 이벤트 보상으로 직접 지급

R18 OFF에서는 R18 전용 후보를 제외하거나 동일 가치의 공통 보상으로 대체한다.
